# 《末日防线》- 剧情大纲

## 一、主线剧情结构

### 📖 四章结构

```
第一章：沦陷 (第 1-10 关)
    ↓
    建立基地，学习防御，首次尸潮
    ↓
第二章：撤离 (第 11-20 关)
    ↓
    基地被毁，转移撤离，发现阴谋
    ↓
第三章：集结 (第 21-30 关)
    ↓
    联合幸存者，对抗军方，揭露真相
    ↓
第四章：总攻 (第 31-40 关)
    ↓
    最终防御战，决战零号病人，双结局
```

---

## 二、详细剧情大纲

### 📘 第一章：沦陷 (第 1-10 关)

**主题：** 生存与建立

**核心剧情：**
> 玩家从昏迷中醒来，城市已沦陷。在幸存者雷克斯引导下建立第一个基地，学习基础防御。经历首次尸潮后，发现丧尸进攻有组织性，背后似乎有人操控。

**关键节点：**

| 关卡 | 事件 | 解锁 |
|------|------|------|
| 第 1 关 | 醒来，被雷克斯救下 | 教程：建造机枪塔 |
| 第 2 关 | 清理基地周围丧尸 | 解锁：探索模式 |
| 第 3 关 | 遇到艾琳，她加入基地 | 解锁：狙击塔，英雄系统 |
| 第 4 关 | 搜索超市，获取第一批物资 | 解锁：农场 |
| 第 5 关 | 胖子首次出现，防线危机 | 解锁：火焰塔 |
| 第 6 关 | 老杰克加入，帮助加固防御 | 解锁：补给塔 |
| 第 7 关 | 发现安布雷拉实验室线索 | 解锁：冰冻塔 |
| 第 8 关 | 尖叫者引来尸潮，艰难防守 | 解锁：散弹塔 |
| 第 9 关 | 影子加入，带来重要情报 | 解锁：探索深层区域 |
| 第 10 关 | **BOSS 战：尸潮领袖** | 解锁：电击塔 |

**章节高潮：**
```
第 10 关：首次 BOSS 战

尸潮领袖出现，指挥大量丧尸进攻基地。
苦战后击败 BOSS，发现它脖子上有安布雷拉的项圈。
艾琳："这不是野生丧尸...有人训练它们！"
雷克斯："安布雷拉...他们到底在搞什么鬼？"
```

**第一章结局：**
> 基地初步建立，但发现了安布雷拉的阴谋线索。艾琳决定调查病毒真相，玩家继续扩建基地，准备应对更大的威胁。

---

### 📗 第二章：撤离 (第 11-20 关)

**主题：** 逃亡与发现

**核心剧情：**
> 巨型丧尸突然出现，摧毁了基地。玩家带领幸存者撤离，沿途建立临时防线。遇到其他幸存者势力，发现军方在秘密研究病毒武器。艾琳获取了部分实验室数据。

**关键节点：**

| 关卡 | 事件 | 解锁 |
|------|------|------|
| 第 11 关 | 巨型丧尸袭击，基地被毁 | 解锁：移动基地车 |
| 第 12 关 | 撤离路上遭遇伏击 | 解锁：自爆丧尸对策 |
| 第 13 关 | 遇到商人联盟，建立贸易 | 解锁：交易系统 |
| 第 14 关 | 帮助掠夺者据点，获取情报 | 选择：救/不救 |
| 第 15 关 | 艾琳获取实验室数据 (配方 1/5) | 解锁：雷达塔 |
| 第 16 关 | 军方直升机出现，侦察基地 | 解锁：防空塔 |
| 第 17 关 | 装甲丧尸群，防线考验 | 解锁：反装甲武器 |
| 第 18 关 | 遇到军方逃兵，得知内幕 | 解锁：新英雄 (可选) |
| 第 19 关 | 建立临时基地，准备反击 | 解锁：导弹塔 |
| 第 20 关 | **BOSS 战：实验体 01** | 解锁：激光塔 |

**章节高潮：**
```
第 20 关：军方实验室突袭

玩家发现军方在秘密研究病毒武器。
实验体 01 失控，玩家被迫迎战。
击败后，格林将军出现：
"你们以为安布雷拉是坏人？我们只是在...保护人类。"
"病毒，是必要的牺牲。"

艾琳："疯子...你们都是疯子！"
```

**第二章结局：**
> 得知军方和安布雷拉都在研究病毒，但目的不同。格林将军警告玩家不要继续调查，否则视为敌人。玩家决定继续前进，揭开真相。

---

### 📙 第三章：集结 (第 21-30 关)

**主题：** 联盟与真相

**核心剧情：**
> 玩家联合各地幸存者，建立联盟。潜入军方基地获取关键数据，发现安布雷拉高层并未死亡，而是在暗中操控一切。艾琳继续收集解药配方，影子的身世逐渐揭开。

**关键节点：**

| 关卡 | 事件 | 解锁 |
|------|------|------|
| 第 21 关 | 建立幸存者联盟 | 解锁：联盟科技 |
| 第 22 关 | 隐形丧尸出现，防线被突破 | 解锁：反隐形措施 |
| 第 23 关 | 解救科学家组织 (医院) | 解锁：医院设施 |
| 第 24 关 | 影子个人任务：潜入安布雷拉分部 | 解锁：影子专属装备 |
| 第 25 关 | 拯救研究员 NPC(配方 2/5) | 解锁：高级研究 |
| 第 26 关 | 军方发动进攻，联盟防御战 | 解锁：联盟支援 |
| 第 27 关 | 反击军方基地，获取数据 | 解锁：配方 3/5 |
| 第 28 关 | 发现安布雷拉日志，真相浮现 | 剧情解锁 |
| 第 29 关 | 准备总攻，集结所有力量 | 解锁：最终武器 |
| 第 30 关 | **BOSS 战：病毒母体** | 解锁：终极技能 |

**章节高潮：**
```
第 30 关：病毒母体

安布雷拉总部地下，巨大的病毒母体在蠕动。
艾琳："这就是...病毒源头！"
击败母体后，发现一份完整的研究日志：

"安布雷拉计划：制造可控丧尸军队，
  净化人类，建立新秩序。
  零号病人是关键，他拥有完整意识..."

雷克斯："所以这一切，都是为了控制？"
艾琳："不...还有更深的阴谋。零号病人，可能不是敌人。"
```

**第三章结局：**
> 发现安布雷拉的真正目的：制造可控丧尸军队，净化人类。但零号病人的身份成谜，艾琳怀疑他可能是受害者而非敌人。玩家集结所有力量，准备最终决战。

---

### 📕 第四章：总攻 (第 31-40 关)

**主题：** 决战与选择

**核心剧情：**
> 丧尸发现玩家基地位置，发动全面总攻。玩家在 10 关中抵御越来越强的尸潮，同时派出英雄小队突袭安布雷拉总部。最终面对零号病人，做出选择：摧毁病毒 (结局 A) 或夺取解药 (结局 B)。

**关键节点：**

| 关卡 | 事件 | 解锁 |
|------|------|------|
| 第 31 关 | 丧尸总攻开始，第一波攻势 | 解锁：紧急动员 |
| 第 32 关 | 尸潮升级，多方向进攻 | 解锁：多线防御 |
| 第 33 关 | 精英丧尸群，防线告急 | 解锁：英雄出击 |
| 第 34 关 | 巨型丧尸群，基地受损 | 解锁：重型武器 |
| 第 35 关 | 智慧丧尸指挥，战术变化 | 解锁：反制战术 |
| 第 36 关 | 英雄小队出发，潜入总部 | 解锁：潜入模式 |
| 第 37 关 | 突破外层防御 | 剧情推进 |
| 第 38 关 | 获取配方 4-5/5，解药完成 | 解锁：真结局条件 |
| 第 39 关 | 最终防线，最猛烈尸潮 | 解锁：全部火力 |
| 第 40 关 | **最终 BOSS：零号病人 + 双结局** | 结局解锁 |

**章节高潮：**
```
第 40 关：最终决战

【场景：安布雷拉总部地下，零号病人被囚禁】

零号病人："你们...是来杀我的？"
艾琳："不，我们是来...理解你的。"

零号病人：
"我不是怪物。我是第一个感染者，也是唯一保持理智的。
 安布雷拉想控制我，军方想杀我。
 但我只想...结束这一切。"

【选择时刻】

选项 A：摧毁病毒 (默认结局)
  → 炸毁实验室，病毒消失，丧尸灭绝
  → 人类重建文明，但感染者全部死亡

选项 B：夺取解药 (真结局，需收集全部配方)
  → 艾琳研发解药，治愈部分丧尸
  → 零号病人牺牲，提供抗体
  → 人类与治愈者共存，新秩序建立
```

**双结局详情：**

### 结局 A：摧毁病毒 (默认)

```
【画面：实验室爆炸，火光冲天】

雷克斯："结束了...终于结束了。"
艾琳："可是...那些能治愈的丧尸..."
雷克斯："他们已经不是人类了。这是必要的牺牲。"

【画面：幸存者重建基地，阳光洒下】

旁白：
"病毒被摧毁，丧尸逐渐灭绝。
 人类开始重建文明，但代价惨重。
 那些本可治愈的生命，成为了历史的尘埃。
 这是胜利...还是遗憾？"

【制作人员名单】
```

**解锁条件：** 直接选择摧毁

---

### 结局 B：解药救赎 (真结局)

```
【画面：艾琳手持解药，零号病人微笑】

零号病人："用我的血...制作解药。
  这是我...最后的贡献。"
艾琳："不...还有别的办法！"
零号病人："没有别的办法了。这是我选择的结局。"

【零号病人牺牲，解药完成】

【画面：治愈的丧尸恢复人类意识，与家人团聚】

旁白：
"解药成功了。部分丧尸恢复了人类意识。
 虽然无法完全逆转，但给了他们第二次机会。

 这不是完美的结局，但...是希望的开始。"

【画面：玩家基地，所有英雄聚集】

雷克斯："你做到了，指挥官。"
艾琳："我们...都做到了。"

【真结局 CG 结束】
```

**解锁条件：**
- ✅ 收集全部 5 个解药配方碎片

### 🎭 选择影响示例

**第 15 关：信任军方**

```
选项 A：信任军方
→ 军方提供支援 (导弹塔 ×2)
→ 后续关卡难度降低
→ 无法解锁真结局线索

选项 B：独立调查
→ 军方敌对，增加难度
→ 解锁隐藏实验室
→ 可获取解药配方碎片
```

**第 25 关：牺牲英雄**

```
场景：英雄小队被困，需要有人断后

选项 A：牺牲一名英雄
→ 其他队员安全返回
→ 该英雄永久死亡
→ 幸存者士气 -20%

选项 B：全员撤退
→ 无人死亡
→ 任务失败，丢失重要数据
→ 解药研究进度 -30%
```
