## 二、背景故事

### 🦠 病毒爆发

**2035 年，T 病毒泄露**

安布雷拉生物科技公司，在地下实验室秘密研究生化武器。一次实验事故导致 T 病毒泄露。

**传播特性：**
- 传播途径：空气、水源、血液
- 潜伏期：2-24 小时
- 感染率：72 小时内全球扩散
- 死亡率：90% 感染，10% 免疫

**感染症状：**
```
初期 (0-2 小时)：发热、呕吐、意识模糊
中期 (2-12 小时)：攻击性增强、瞳孔放大、皮肤灰白
后期 (12-24 小时)：心跳停止、大脑坏死、嗜血本能
变异期 (7 天后)：部分个体产生变异，获得特殊能力
```

### 🏚️ 文明崩塌

**72 小时沦陷**

| 时间 | 事件 |
|------|------|
| 第 1 小时 | 实验室泄露，内部人员感染 |
| 第 6 小时 | 病毒扩散至周边社区 |
| 第 24 小时 | 城市沦陷，军队介入 |
| 第 48 小时 | 军队防线崩溃 |
| 第 72 小时 | 全球主要城市沦陷 |
| 第 7 天 | 文明秩序完全崩塌 |

### 🏕️ 幸存者基地

**少数免疫者建立聚集地**

玩家扮演一名基地指挥官，负责：
- 建造和升级防御塔
- 指挥幸存者英雄
- 收集资源维持基地
- 探索废墟寻找物资
- 揭开病毒背后的阴谋

## 四、势力设定

### 🧟 丧尸阵营

**蜂巢意识**

丧尸通过信息素交流，形成蜂巢意识：
- 普通丧尸：无意识，跟随气味
- 精英丧尸：部分意识，可指挥普通丧尸
- BOSS 丧尸：完整意识，制定战术

**尸潮机制**
- 月圆之夜：丧尸活跃度 +50%
- 雨天：丧尸移动速度 -20%
- 高温：丧尸腐化加速，自燃概率增加

### 👥 幸存者阵营

**主要势力**

| 势力 | 位置 | 态度 | 说明 |
|------|------|------|------|
| **玩家基地** | 城市郊区 | 友好 | 玩家控制的基地 |
| **军方避难所** | 军事基地 | 中立→敌对 | 研究病毒武器 |
| **商人联盟** | 购物中心 | 中立 | 以物易物，唯利是图 |
| **科学家组织** | 医院 | 友好 | 寻找解药 |
| **掠夺者** | 高速公路 | 敌对 | 抢劫幸存者 |

### 🧪 安布雷拉公司

**幕后黑手**

| 信息 | 内容 |
|------|------|
| **成立时间** | 1998 年 |
| **表面业务** | 医药、化妆品 |
| **秘密研究** | 生化武器、病毒实验 |
| **现状** | 总部沦陷，高层失踪 |

## 六、核心冲突

### 🎯 玩家目标

**短期目标**
- 生存：抵御每日尸潮
- 发展：扩建基地，升级防御
- 探索：收集资源，解锁新区域

**中期目标**
- 揭开病毒真相
- 建立幸存者联盟
- 对抗军方势力

**长期目标**
- 摧毁/夺取病毒研究
- 研发解药 (隐藏线)
- 重建人类文明

### ⚔️ 主要矛盾

| 矛盾 | 双方 | 核心 |
|------|------|------|
| 人类 vs 丧尸 | 生存 vs 毁灭 | 资源争夺 |
| 玩家 vs 军方 | 真相 vs 掩盖 | 病毒研究 |
| 幸存者 vs 掠夺者 | 秩序 vs 混乱 | 道德选择 |
| 解药派 vs 摧毁派 | 治愈 vs 灭绝 | 最终选择 |
