- **病毒爆发**：全屏伤害，需要躲避
- **吞噬**：直接摧毁并"吃掉"防御塔 (恢复 5% 血量)

**攻略：**
1. 多线防守，防止瞬移偷袭
2. 被控制的塔用修复塔净化
3. 病毒爆发时英雄躲避
4. 真结局需要艾琳在场
