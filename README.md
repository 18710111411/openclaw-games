# 🧟 末日防线 (Last Stand)

> 丧尸末日生存塔防游戏 - 建造防御塔，指挥英雄，在丧尸潮中生存！

[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)
[![Unity](https://img.shields.io/badge/Unity-2022+-black?logo=unity)](https://unity.com/)
[![Platform](https://img.shields.io/badge/platform-Windows%20%7C%20macOS%20%7C%20Android%20%7C%20iOS-lightgrey)]()

---

## 🎮 游戏特色

### 核心玩法

- **🏰 塔防 + 探索双模式** - 白天探索废墟收集资源，夜晚抵御丧尸潮
- **🦸 英雄系统** - 5 位核心英雄，每位都有独特技能和背景故事
- **🧟 多样丧尸** - 13 种丧尸类型，从普通感染者到 BOSS 级威胁
- **🏗️ 12 种防御塔** - 物理/元素/特殊功能塔，策略搭配
- **📖 动态任务系统** - NPC 势力也在行动，任务结果随选择变化
- **🎭 双结局** - 摧毁病毒 or 解药救赎，你的选择决定结局
