## 👤 主角

### 基本信息

| 项目 | 设定 |
|------|------|
| **穿越前姓名** | 李凡（暂定，可改） |
| **穿越后姓名**  | 李执事（NPC 身份） |
| **年龄** | 穿越前 28 岁 / 穿越后外貌 25 岁 |
| **性别** | 男 |
| **身份** | 青云宗·外门任务堂执事 |
| **境界** | 开局筑基初期 |
| **灵根** | 四灵根（中等偏下） |

### 性格特点

| 特点 | 描述 |
|------|------|
| **务实** | 不整虚的，能变强就行 |
| **精打细算** | 每个任务都要算收益 |
| **护短** | 我的人只有我能欺负 |
| **有底线** | 不坑玩家，双赢 |
| **肝帝思维** | 总觉得不干活就亏了 |

### 外貌描写

```
一袭青色执事袍，面容清秀，眼神锐利。
说话时习惯性地摩挲下巴，思考时眉头微皱。
腰间挂着任务令牌和储物袋。
```

### 背景故事

**穿越前：**
- 28 岁程序员，业余全服第一肝帝
- 连续肝了 72 小时后猝死
- 游戏 ID：「肝帝本帝」
- 成就：全职业满级，首通所有副本

**穿越后：**
- 醒来成为青云宗任务堂执事
- 发现可以发布任务给玩家
- 玩家完成任务，自己获得双倍奖励
- 从此走上"让玩家代肝"的巅峰之路

### 成长弧线

```
初期：小心翼翼试探金手指
      ↓
中期：系统化运营，建立玩家公会
      ↓
后期：掌控全局，玩家大军征战万界
      ↓
结局：成仙，但选择继续"游戏"
```

### 口头禅

- "这个任务报酬丰厚，谁来接？"
- "放心，跟着我有肉吃"
- "这个我熟，当年我肝了 XXX 次"
- "不急，稳一点"

### 核心技能

| 技能 | 等级 | 说明 |
|------|------|------|
| 任务发布 | 满级 | 可发布任何难度任务 |
| 双倍返利 | 满级 | 玩家完成，主角双倍奖励 |
| 返利暴击 | 1% 概率 | 小概率触发 10 倍返利 |
| 任务追踪 | 满级 | 实时查看任务进度 |
| 玩家评价 | 满级 | 查看玩家对任务评分 |

## 🍀 配角 5：郝运（逗比福星）⚠️ 新增

### 基本信息

| 项目 | 设定 |
|------|------|
| **姓名** | 郝运（谐音"好运"） |
| **年龄** | 19 岁 |
| **修为** | 练气三层 |
| **灵根** | 五灵根（废柴资质） |
| **身份** | 外门弟子 |
| **出场章节** | **第 5 章起**（前十章少量出场 2-3 次） |

### 性格特点

| 特点 | 描述 |
|------|------|
| **憨厚** | 老实巴交，说话直 |
| **逗比** | 经常说错话/闹笑话 |
| **幸运** | 运气爆棚，误打误撞立功 |
| **不降智** | 关键时刻靠谱 |

### 作用

- 调节气氛，缓解紧张剧情
- 误打误撞帮助主角
- 关键时刻意外立功
- 增加故事趣味性

### 经典台词

- "俺也不知道咋回事，就……就这样了"
- "俺就是随便走走，谁知道捡到个宝贝……"
- "李师兄，俺是不是又说错话了？"

### 出场计划

| 章节范围 | 出场次数 | 说明 |
|---------|---------|------|
| 第 5-10 章 | 2-3 次 | 少量出场，慢慢引入 |
| 第 11-20 章 | 5-8 次 | 逐渐增多 |
| 第 21 章起 | 常规出场 | 核心配角之一 |

### 经典桥段示例

**第 5 章首次出场：**
```
李凡正在后山寻找火灵草，突然听到一阵窸窸窣窣的声音。

"谁？"他警惕地转身，手中法诀凝聚。

一个身穿破旧道袍的少年从草丛里钻出来，手里捧着一株散发着红色光芒的草。

"俺……俺不是故意的。"少年憨厚地笑了笑，挠挠头，"俺就是随便走走，谁知道这草自己跑到俺手里了……"

李凡定睛一看，差点没站稳。

那不是火灵草是什么？

而且看这成色，至少是百年份的！

"你……怎么找到的？"李凡声音有些干涩。

"俺也不知道咋回事。"少年一脸无辜，"俺就是觉得这地方暖和，想躺会儿，结果一摸，就摸到这草了……"

李凡沉默了。

他辛辛苦苦找了半天，用系统才知道具体位置。

这货随便走走，就碰到了？

这就是运气么……
```

**第 8 章误打误撞立功：**
```
赵虎带人围住李凡，冷笑一声："李凡，今天看你往哪跑！"

就在这千钧一发之际，一个身影从旁边的小路冒出来。

"哎呀，俺是不是走错路了？"郝运一脸茫然，手里捧着一堆灵果，"俺就是想找点吃的，谁知道走到这来了……"

赵虎脸色一沉："滚开！没你的事！"

"哦哦，俺这就走。"郝运点点头，转身就走。

可他刚转身，脚下一滑，整个人向前扑去。

他手里的灵果散落一地，咕噜噜滚向赵虎等人。

"该死！"赵虎闪身躲避。

可谁也没想到，这些灵果遇到灵力，竟然爆炸了！

轰！

赵虎等人被炸得灰头土脸，狼狈不堪。

李凡趁机脱身。

事后，郝运一脸无辜："俺真不知道这些果子会爆炸……俺就是洗了洗，想留着慢慢吃……"

李凡深深看了他一眼，心中暗自思忖。

这运气，也太离谱了吧？
```

## ⚠️ 反派塑造原则（老板强调）

| 原则 | 说明 |
|------|------|
| **不降智** | 反派有合理动机，不犯低级错误 |
| **任务丰满** | 有背景故事，有成长弧线 |
| **立场鲜明** | 不是为坏而坏，有自己的坚持 |
| **值得尊重** | 即使是敌人，也有可敬之处 |
