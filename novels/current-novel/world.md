# 世界观设定

## 🌍 世界背景

### 游戏名称
**《仙途 Online》** - 沉浸式修仙网游

### 世界本质
- **表面：** 一款划时代的沉浸式修仙网游
- **真相：** 真实存在的修仙世界，被"天道"包装成游戏
- **NPC：** 原本是世界原住民，部分已觉醒自我意识
- **玩家：** 来自现实世界，情感能量是"天道"的燃料

### 时间线

| 时间点 | 事件 |
|--------|------|
| 远古 | 天道创造世界，生灵繁衍 |
| 10 年前 | 第一名穿越者降临（老酒鬼） |
| 1 年前 | 主角穿越，成为 NPC |
| 公测当天 | 玩家大规模降临 |
| 现在 | 故事开始 |

## 👥 社会结构

### 修仙界势力

```
                    ┌─────────────┐
                    │   天道      │
                    │  (幕后 BOSS)│
                    └──────┬──────┘
                           │
        ┌──────────────────┼──────────────────┐
        │                  │                  │
  ┌─────┴─────┐    ┌──────┴──────┐    ┌─────┴─────┐
  │  正道联盟 │    │  魔道联盟   │    │  中立势力 │
  │  (7 大宗)  │    │  (5 大宗)   │    │  (散修等) │
  └───────────┘    └─────────────┘    └───────────┘
```

### 玩家势力

| 势力类型 | 代表 | 特点 |
|----------|------|------|
| **大型公会** | 打工者联盟 | 数万人，组织严密 |
| **财团势力** | 腾达集团 | 资本支持，目的不纯 |
| **散人玩家** | 各种 ID | 自由，易被利用 |
| **工作室** | 职业代练 | 盈利为目的 |

### 核心冲突（老板设定）

| 冲突类型 | 双方 | 矛盾点 |
|----------|------|--------|
| **NPC vs 玩家** | 觉醒 NPC vs 玩家入侵 | 生存空间、资源争夺 |
| **门派争斗** | 正道 vs 魔道 | 理念不同、资源分配 |
| **财团博弈** | 各大财团之间 | 利益分配、控制权 |
| **开发者 vs 玩家** | 天道/创造者 vs 玩家 | 操控与反操控 |

## ✨ 特殊设定

### 金手指系统

**【双倍返利系统】** - 主角专属

| 功能 | 说明 |
|------|------|
| 任务发布 | 可发布任何任务给玩家 |
| 双倍返利 | 玩家完成，主角获得双倍奖励 |
| 返利暴击 | 1% 概率触发 10 倍返利 |
| 任务追踪 | 实时查看任务进度 |
| 玩家评价 | 查看玩家对任务评分 |

### 玩家系统

| 功能 | 说明 |
|------|------|
| 复活机制 | 死亡后在复活点重生 |
| 等级系统 | 1-100 级，对应修仙境界 |
| 任务系统 | 可接取 NPC 发布的任务 |
| 交易行 | 玩家之间自由交易 |
| 论坛系统 | 玩家交流、攻略分享 |

### 觉醒机制

| 类型 | 说明 |
|------|------|
| **NPC 觉醒** | 部分 NPC 产生自我意识 |
| **穿越者** | 从现实世界穿越而来 |
| **异常数据** | 系统无法解释的存在 |
