### 穿越前

| 项目 | 设定 |

| **姓名** | 待定 |

### 穿越后

| 项目 | 设定 |
|------|------|
| **NPC 身份** | 青云宗·外门任务堂执事 |
| **境界** | 开局筑基初期（有下降风险） |
| **金手指** | 【双倍返利系统】 |
| **性格** | 继续肝帝思维，但发现可以"让玩家代肝" |

### 金手指详解

**【双倍返利系统】**

```
功能 1：发布任务
  - 可发布任何任务给玩家
  - 任务奖励由主角设定
  - 玩家完成后，主角获得双倍奖励

功能 2：任务追踪
  - 实时查看任务进度
  - 可远程协助玩家（消耗灵力）
  - 可调整任务难度

功能 3：返利暴击
  - 小概率触发 10 倍返利
  - 概率：1%
  - 可提升（消耗特殊道具）

功能 4：玩家评价系统
  - 玩家对任务评分
  - 高评分吸引更多玩家
  - 低评分降低吸引力
```

**示例：**
```
主角发布：采集 10 株灵草
玩家奖励：100 灵石 + 100 经验
主角获得：200 灵石 + 200 经验（双倍）
触发暴击：2000 灵石 + 2000 经验（10 倍）
```
