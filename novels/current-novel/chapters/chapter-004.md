# 第四章：废弃矿洞的秘密

// 进入矿洞

第二天清晨，李凡早早起床。窗外天色微亮，晨雾还未散去，空气中带着淡淡的草木清香。

他仔细检查了一下自己的装备，手指逐一抚过储物袋中的物品。回灵丹还有十颗，金疮药也是十颗，金刚符仅剩一张，下品灵石三十五块，再加上原主遗留的那个储物袋——这就是他全部的家当了。

"差不多了。"李凡低声自语，眼中闪过一丝决然。他将储物袋背在身上，袋口系紧，然后推开房门，离开宗门，往后山走去。

昨晚他只睡了两个时辰，辗转反侧难以入眠。剩下的时间，他一直在思考寿元的问题，脑海中不断盘算着系统的限制。

"系统每天免费额度只有三次。"李凡眉头紧锁，心中暗自忧虑，"超出后，每次消耗一天寿元。昨天一次就花了十天。如果再来几次……"

他不敢想下去，胸口仿佛压了一块巨石。

"必须找到补充寿元的方法。"李凡咬了咬牙，眼神坚定，"或者，尽量少用系统。"

但问题是，在这个危机四伏的修仙世界，不用系统，他拿什么和那些天才弟子争？拿什么和那些老怪物斗？

"难啊……"李凡长叹一声，脸上写满了无奈。他抬头望向远处的山峦，晨雾缭绕，宛如仙境，却暗藏杀机。

后山是青云宗外门弟子的活动区域，平时有不少弟子在这里猎杀妖兽、采集灵草。空气中偶尔传来几声鸟鸣，清脆悦耳。但废弃矿洞所在的位置，比较偏僻，人迹罕至。

根据任务玉简的提示，矿洞在后山西北角，距离外门大约十里。李凡御风而行，练气四层的修为，速度还算不错。风声在耳边呼啸，衣袂飘飘。

大约半个时辰后，他来到了矿洞入口。这是一个巨大的山洞，洞口布满青苔，绿得发黑，看起来已经废弃很久了。潮湿的气息扑面而来，带着泥土和岩石的味道。

洞口立着一块石碑，上面刻着几个大字，字迹斑驳，依稀可辨：

青云矿洞，闲人免进

"就是这里了。"李凡深吸一口气，空气中弥漫着腐朽的气息。他激活了金刚符，一层淡淡的金色光晕笼罩在他体表，这是金刚符的效果，可以抵挡一次致命攻击。

"进去吧。"他取出一个照明符，扔进洞内。照明符亮起，发出柔和的白光，照亮了前方的路。

矿洞内很安静，只有水滴落下的声音，滴答，滴答，在空旷的洞中回荡。李凡小心翼翼地往前走，神识外放，警惕着周围的动静。脚下的碎石发出轻微的咯吱声。

练气期的修士，神识还不强，只能覆盖周围十丈的范围。但对付一般的危险，足够了。洞壁上的岩石湿漉漉的，偶尔能看到一些不知名的苔藓，散发着微弱的荧光。

// 第一只妖兽

走了大约一炷香的时间，李凡突然停下了脚步，眉头一皱。

"有动静。"他低声说道，眼中闪过一丝警惕。

他感觉到，前方有灵力波动，而且，不止一个。那波动时强时弱，像是在移动。

李凡躲到一块岩石后面，悄悄观察。岩石表面冰凉粗糙，他屏住呼吸，尽量不发出任何声音。

很快，他看到了动静的来源。三只妖兽，正围在一起，似乎在分食什么。空气中传来一股血腥味。

这些妖兽长得像狼，但体型更大，身上散发着淡淡的黑气，双眼泛着幽绿的光芒。

"黑风狼，练气五层妖兽。"李凡心中一凛，手心渗出冷汗。

黑风狼是群居妖兽，通常三五只一起行动。它们的速度极快，擅长偷袭。对于练气四层的李凡来说，一只黑风狼还能应付，但三只……

"得想个办法。"李凡眯起眼睛，大脑飞速运转。他没有贸然出手，而是继续观察，寻找机会。

很快，他发现了一个细节。这三只黑风狼中，有一只体型明显较小，应该是狼群中的弱者。而另外两只，体型较大，应该是头狼和配偶。

"先解决弱的。"李凡做出了决定，眼中闪过一丝狠厉。

李凡深吸一口气，体内灵力开始涌动。他双手迅速结印，指尖在胸前划过一道玄奥的轨迹。练气四层的灵力在经脉中快速流转，汇聚于掌心。

"火球术！"

一声低喝，赤红色的火球在他掌心凝聚，表面火焰跳动，散发出灼热的气息。周围的空气都因高温而扭曲，热浪扑面而来。

李凡眼神一凝，手腕一抖，火球呼啸而出，精准地射向那只较小的黑风狼。火球划破空气，带起一道赤红色的尾焰。

火球精准地击中了黑风狼的头部，发出一声闷响。

"嗷！"黑风狼惨叫一声，声音凄厉。它倒在地上，挣扎了几下，四肢抽搐，不动了。空气中弥漫着焦糊的味道，皮毛被烧焦，冒着缕缕青烟。

"成功了！"李凡心中一喜，嘴角微微上扬。

火球术消耗了他三十五点灵力，但效果显著。用来偷袭练气五层的妖兽，正好合适。

但另外两只黑风狼被惊动了。它们转头看向李凡的方向，眼中满是愤怒，幽绿的光芒更盛。

"吼！"两只黑风狼同时低吼，声音震耳欲聋。它们后腿一蹬，扑了过来，速度快如闪电。黑色的气流在它们周身环绕，那是黑风狼的天赋妖力。

李凡不敢大意，双脚迅速后退，同时双手再次结印。他的额头上渗出细密的汗珠，灵力在体内快速流转。

"水箭术！"

清澈的水蓝色灵力在他掌心凝聚，化作一支晶莹剔透的水箭。水箭表面流转着淡淡的光晕，周围的水汽凝结成细小的水珠，在空中悬浮。

李凡手腕一抖，水箭呼啸而出，迎向其中一只黑风狼。

黑风狼反应极快，身体一扭，险之又险地避开了水箭。水箭擦着它的皮毛飞过，击中后方的岩壁，发出一声脆响，岩石被击穿一个小洞。

"速度太快了！"李凡心中一凛。

两只黑风狼已经扑到面前，腥风扑面而来。李凡只能被迫近身闪避，身形灵活地在两只妖兽之间穿梭。

// 苦战

黑风狼的速度很快，眨眼间就冲到了李凡面前。腥风扑面而来，李凡几乎能闻到它口中的血腥味。

李凡双手迅速结印，体内剩余的灵力再次涌动。他的脸色有些苍白，连续施展法术让他的灵力消耗很大。

"火球术！"

又一颗火球凝聚而出，但比之前的小了一些，火焰也没那么旺盛。李凡咬牙，将火球射向左边那只黑风狼。

火球击中黑风狼的前腿，发出一声闷响。黑风狼吃痛，发出一声低沉的呜咽，动作缓了一缓。

但右边的黑风狼趁机扑了上来，一口咬向李凡的肩膀。李凡躲闪不及，被咬中了手臂。

"嘶！"剧烈的疼痛传来，李凡倒吸一口凉气，脸色瞬间苍白。鲜血从伤口涌出，染红了衣袖。

他立刻后退几步，与黑风狼拉开距离，同时从储物袋中取出一颗金疮药，塞入口中。药丸入口即化，伤口开始止血，带来一丝清凉。

"不能再拖了。"李凡心中焦急，呼吸变得急促。他还有另一只黑风狼没解决，灵力也在快速消耗。

刚才两次火球术消耗了他七十点灵力，一次水箭术消耗了三十点，他现在只剩下不到二十点灵力。

"拼了！"李凡咬牙，眼中闪过一丝决绝。

他深吸一口气，强行调动体内剩余的灵力。双手再次结印，指尖因为灵力不足而微微颤抖。

"火球术！"

一颗比之前小得多的火球在掌心凝聚，火焰微弱，像是随时会熄灭。但李凡没有犹豫，手腕一抖，火球射向那只受伤的黑风狼。

火球精准地击中了黑风狼的眼睛。黑风狼发出一声凄厉的惨叫，原地打转，失去了方向感。

"就是现在！"

李凡眼中闪过一丝狠厉，身形如电，欺身而上。他取出长剑，剑身寒光闪闪，用尽全身力气刺向黑风狼的咽喉。

噗嗤！

长剑没入黑风狼的咽喉，鲜血喷涌而出。黑风狼挣扎了几下，不动了。

"呼……"李凡长舒一口气，但还没等他放松，另一只黑风狼又扑了过来。

这只黑风狼看到同伴被杀，眼中满是愤怒，周身黑气更盛。它张开血盆大口，咬向李凡的喉咙。

李凡想要躲闪，但体力已经透支，动作慢了一拍。

就在这千钧一发之际，李凡的脑海中突然闪过一个念头。他强提一口气，双手最后一次结印。

"水箭术！"

一颗水箭在极近的距离凝聚，虽然威力不大，但胜在距离近。水箭精准地射入黑风狼张开的口中，从后脑穿出。

黑风狼的身体僵住了，眼中的光芒迅速消散。它轰然倒地，激起一片灰尘。

"赢了……"

李凡瘫坐在地上，后背全是冷汗。这一战，他赢了，但赢得并不轻松。

手臂上的伤口还在流血，灵力也消耗得一干二净。他感觉全身乏力，手臂颤抖。

"先疗伤。"李凡服用了一颗金疮药，又打坐恢复灵力。他闭上眼睛，运转《青云诀》，周围的灵力缓缓涌入体内。

大约半个时辰后，他的状态恢复了不少。伤口已经结痂，灵力也恢复了七成。

"继续前进。"李凡站起身，拍了拍身上的灰尘，继续往矿洞深处走去。脚步声在空旷的矿洞中回荡。

// 矿洞深处

越往深处走，矿洞内的灵力波动就越强烈。李凡能感觉到，前方有什么东西在散发着灵力，那波动时强时弱，像是在呼吸。

"难道是灵矿？"他心中一动，眼中闪过一丝期待。

青云矿洞之所以废弃，就是因为当年的灵矿被开采一空。但如果还有残留的灵矿，那也是一笔不小的财富。

又走了一炷香的时间，李凡来到了一个巨大的矿洞大厅。大厅中央，有一个石台，石台上放着一个盒子。

盒子散发着淡淡的灵力波动，显然不是凡物。那波动柔和而神秘，让人忍不住想要靠近。

"这就是异常灵力波动的来源？"李凡小心翼翼地靠近石台，脚步放得很轻。

他没有立刻打开盒子，而是先观察周围。大厅内很安静，没有妖兽的踪迹，只有他自己的呼吸声。

但李凡总觉得，有什么地方不对劲。太安静了，安静得有些诡异。

"太安静了。"他低声自语，眉头紧锁，"一路走来，遇到了三只黑风狼。但这里，什么都没有。"

这不正常。如果这里有宝物，应该会有妖兽守护才对。除非……

"这是个陷阱。"李凡心中一凛，背后升起一股寒意。

他立刻后退，远离石台，脚步急促。就在他后退的瞬间，大厅四周突然亮起了符文，蓝色的光芒闪烁。

一个巨大的阵法被激活了，符文流转，发出嗡嗡的声响。

"果然！"李凡脸色一变，心中暗骂。他想跑，但已经晚了。

阵法激活后，整个大厅被封闭了。出口被堵住，一道光幕落下，将李凡困在了里面。

"该死！"李凡暗骂一声，拳头紧握。他尝试攻击阵法，但阵法纹丝不动，灵力撞击在光幕上，被反弹回来。

这个阵法的等级，远超他的修为。他感觉自己的灵力在阵法面前，如同蝼蚁撼树。

"冷静，一定有办法破解。"李凡强迫自己冷静下来，深吸几口气。他开始观察阵法，寻找破绽。

阵法上有十二个符文，分别对应十二个时辰。那些符文闪烁着微弱的光芒，按照某种规律流转。

"子、丑、寅、卯……"李凡一个一个地看过去，眼睛一眨不眨。

很快，他发现了一个细节。其中一个符文，亮度比其他符文暗了一些，像是接触不良。

"那个符文，可能是阵眼。"李凡心中一动，眼中闪过一丝希望。

他取出长剑，注入灵力，刺向那个符文。剑尖与符文接触的瞬间，灵力爆发。

"轰！"阵法剧烈震动，那个符文破碎了，化作点点光芒消散。

紧接着，整个阵法开始瓦解，符文一个接一个地熄灭。几秒钟后，阵法消失，出口重新打开。

"成功了！"李凡心中一喜，脸上露出笑容。他没想到，自己真的能破解这个阵法。

"可能是我运气好，那个符文本来就是松动的。"李凡没有多想，走到石台前，打开了盒子。

盒盖缓缓开启，发出轻微的吱呀声。盒子里，放着三样东西：一本古籍，一瓶丹药，一块玉佩。

李凡先拿起古籍，翻开第一页。纸张泛黄，字迹古朴。

"《青云诀·进阶篇》？"他微微失望，眉头轻皱。不是寿元相关的功法。

但他还是继续看了下去。这是一本修炼功法，比外门弟子学的《青云诀》更深奥。

"青云诀进阶篇，修炼效率是基础篇的两倍。但要求修为达到练气期才能修炼。"李凡摇摇头，将古籍收好。

"现在用不了。不过，可以留着以后用。"他收起古籍，继续查看盒子里的其他东西。

那瓶丹药，标签上写着：聚灵丹，上品，十颗。瓶身温润，散发着淡淡的药香。

聚灵丹可以快速恢复灵力，比回灵丹的效果更好。

"好东西！"李凡眼中一亮，嘴角上扬。他收下聚灵丹，这对他现在的战斗很有帮助。

最后，他拿起那块玉佩。玉佩温润如玉，触手生温，上刻着一个字：李。

"这是……"李凡心中一动，手指轻轻摩挲着玉佩表面。

这个字，和他的姓氏一样。他注入灵力，玉佩亮起，浮现出一行字，光芒柔和。

"李氏后人，若见此玉佩，可前往中州李家，领取传承。"

"中州李家？"李凡皱眉，眼中满是疑惑。

原主的记忆里，没有这个信息。原主是孤儿，被青云宗收养，不知道自己的身世。

但这块玉佩……

"难道原主不是孤儿？"李凡喃喃自语，"他的父母，是中州李家的？"

李凡想不通，摇摇头，将玉佩收好。但他知道，这块玉佩很重要。

他收起玉佩，准备离开。就在这时，系统的声音响起，在他脑海中回荡：

【检测到宿主完成任务】

【任务名称：探索废弃矿洞】

【任务奖励：一百贡献点加两百灵石】

【额外奖励：寿元加三十天（首次完成 B 级任务）】

李凡一愣，眼中闪过一丝惊喜。

"任务完成了？"他在心中问道。

【是的】

【任务目标：调查异常灵力波动】

【调查结果：发现上古修士遗留的宝物】

【任务完成度：百分之百】

李凡沉默了一会儿，心中五味杂陈。

"原来，完成任务可以获得寿元奖励。"他低声说道，"但这只是额外奖励，不是每次都有。而且，三十天寿元，看起来多，实际上……"

他想起昨天消耗的十天，心中一紧。

"一次临时提示就花了十天。如果多来几次，三十天也不够。所以，这只是一个暂时的缓解。真正的解决方法，还在后面。"

李凡收起思绪，转身离开。

"先回去交任务。"

他离开矿洞，返回宗门。阳光洒在身上，温暖而明亮，与矿洞内的阴冷形成鲜明对比。

// 返回宗门

回到外门，李凡先去任务堂交了任务。任务堂内人来人往，热闹非凡。

周执事看到他，有些惊讶，挑了挑眉："这么快就完成了？"

"侥幸。"李凡谦虚道，微微躬身。

"嗯，不错。"周执事点点头，眼中带着赞赏，"奖励已经发放到你的令牌里了。"

李凡查看令牌，果然多了一百贡献点和两百灵石。令牌微微发热，显示着新的数值。

"多谢执事。"李凡拱手道谢。

"不用谢。"周执事顿了顿，眼中闪过一丝意味深长，"对了，下个月有一场内门弟子选拔，你要不要参加？"

"内门弟子选拔？"李凡心中一动，眼中闪过一丝期待。

"是的。"周执事说道，"外门弟子中，修为达到练气五层的，都可以参加。通过选拔，就能成为内门弟子。内门弟子，能学到更好的功法，有更多的资源。"

他顿了顿，继续说道："你现在的修为是练气四层，还差一点。但如果能在下个月前突破到练气五层，就可以参加。"

李凡心中一凛，拳头不自觉地握紧。内门弟子，这正是他的目标。成为内门弟子，就有资格竞选任务堂执事了。

"多谢执事告知，弟子会努力的。"李凡认真地说道，眼中满是坚定。

"嗯，好好修炼。"周执事摆摆手，脸上露出鼓励的笑容，"我看好你。"

李凡离开任务堂，心中有了计划。阳光照在脸上，暖洋洋的。

"下个月内门选拔，必须参加。"他暗自下定决心，"要参加，就必须突破到练气五层。现在离下个月还有二十天。时间紧迫，得抓紧修炼。"

但还有一个问题，始终萦绕在他心头，如同阴云不散。

"寿元……"李凡低声念叨，"虽然获得了三十天奖励，但这只是暂时的。我必须找到稳定的补充方法。或者，尽量少用系统。"

李凡回到自己的小屋，取出《青云诀·进阶篇》，开始研究。虽然现在用不了，但可以提前熟悉。书页翻动的声音在安静的小屋中格外清晰。

"二十天时间，突破练气五层。然后参加内门选拔。成为内门弟子后，资源更多，也许能找到寿元相关的线索。"

李凡握紧拳头，眼中燃起斗志。

"一步一步来。问题总会解决的。"

……

而在矿洞深处，一个黑影出现在大厅里。那黑影如同鬼魅，悄无声息。

他看着破碎的阵法，脸色阴沉，眼中闪过一丝怒意。

"有人来过。"黑影低声说道，声音沙哑，"盒子被取走了。是谁？"

黑影沉默了一会儿，取出传音符。传音符在他手中微微发光。

"主公，矿洞的宝物被人取走了。"他对着传音符说道，"是……李氏玉佩。属下立刻追查。"

传音符消散，化作一道流光飞走。黑影也消失了，如同从未出现过。

矿洞重新恢复了平静，仿佛什么都没发生过。只有地上的痕迹，证明曾经有人来过。

但在某个角落，老酒鬼望着矿洞的方向，喃喃自语。他倚靠在一棵老树下，手中拿着酒壶。

"他进去了。也出来了。"老酒鬼喝了一口酒，酒香四溢，"希望他能找到答案。寿元的问题……我也找了三十年，至今没有头绪。"

老酒鬼又喝了一口酒，眼神有些迷离。

"小子，加油吧。这条路，不好走。"
