# 第五章：内门选拔，我要赢

// 二十天计划

回到小屋后，李凡立刻开始制定计划。他坐在桌前，手指轻轻敲击着桌面，发出有节奏的声响。

距离内门选拔，还有二十天。窗外的阳光透过窗棂洒进来，在桌面上投下斑驳的光影。

他要在这二十天内，从练气四层突破到练气五层。这个目标如同大山般压在他心头。

李凡闭上眼睛，在心中呼唤系统。片刻后，他的状态浮现在脑海中：

宿主：李凡
境界：练气四层，进度四比一百二十
经验值：四比一百二十
寿元：九十九年三百五十五天
灵石：二百三十五，任务奖励两百
贡献点：二百零三
今日系统额度：三比三，已用完

李凡看着自己的状态，开始计算，眉头微皱。

"从练气四层到五层，需要一百二十点经验值。"他低声念叨，手指在桌上划动，"我现在的经验值是四点，还差一百一十六点。做任务是最快的方式。"

他打开任务玉简，查看可接取的任务。玉简散发着淡淡的蓝光，触手温润。

但这次，他没有用系统的预知功能。因为今天的免费额度已经用完了。而且，他也不想再消耗寿元。

"只能靠自己了。"李凡叹了口气，眼中闪过一丝无奈。

他仔细浏览任务列表，寻找性价比高的任务。任务玉简上的文字一个个闪过，他的目光专注而认真。

很快，他锁定了几个任务，眼中一亮：

采集灵草二，奖励十五贡献点加三十灵石，经验值八点，耗时两小时
清理妖兽二，奖励二十贡献点加四十灵石，经验值十二点，耗时三小时
护送物资，奖励二十五贡献点加五十灵石，经验值十五点，耗时四小时

"这几个任务不错，经验值奖励高。"李凡嘴角微微上扬，心中有了计较。

李凡决定，今天先做这三个任务。预计能获得三十五点经验值。再加上修炼获得的经验值，一天应该能拿四十点左右。

"一百一十六除以四十，大约需要三天。"他掰着手指计算，"三天后，我就能突破到练气五层。然后还有十七天时间，可以用来准备内门选拔。"

计划很完美。李凡站起身，拍了拍衣服，眼中满是斗志。

"出发！"他推门而出，去接取任务。晨风吹在脸上，带着一丝凉意。

// 任务日常

接下来的三天，李凡过得很充实。每天清晨，第一缕阳光刚照进小屋，他就起床了。

每天早上，他起床修炼一个时辰，然后出门做任务。清晨的空气中带着露水的气息，清新宜人。

晚上回来，继续修炼《青云诀》。月光透过窗户洒在身上，静谧而安详。

三天下来，他的收获不小。身体虽然疲惫，但眼神却愈发明亮。

李凡再次查看自己的状态：

宿主：李凡
境界：练气四层，进度一百一十八比一百二十
经验值：一百一十八比一百二十
寿元：九十九年三百五十八天，加三天
灵石：四百一十
贡献点：三百二十

"还差两点经验值！"李凡眼中闪过一丝兴奋，拳头不自觉地握紧。

只要再完成一个小任务，他就能突破了。那种期待感让他的心跳加速。

他来到任务堂，寻找最简单的任务。任务堂内人来人往，热闹非凡。

"帮助外门食堂搬运食材，奖励一贡献点加二经验值。"李凡看着任务板上的信息，眼中一亮。

"就这个了！"他伸手取下任务牌，任务牌入手微凉。

李凡接取任务，去食堂帮忙。食堂位于外门东侧，远远就能闻到饭菜的香味。

工作很简单，就是把食材从仓库搬到厨房。一袋袋米面蔬菜，沉甸甸的，压在身上。

李凡干得很认真，额头上渗出细密的汗珠。他一袋一袋地搬运，动作麻利。

两个时辰后，任务完成。他擦了擦汗，长舒一口气。

任务完成！
获得：一贡献点加二灵石加二经验值
经验值：一百二十比一百二十，已满
检测到经验值已满，是否突破？

"突破！"李凡毫不犹豫，声音坚定有力。

轰！

一股强大的灵力从他体内涌出，周围的空气都为之震动。衣袂无风自动，猎猎作响。

练气五层！

突破成功！
当前境界：练气五层
灵力：一百五十比一百五十

李凡感受着体内澎湃的灵力，心中满是喜悦。那灵力如同江河般在体内奔涌，强大而充盈。

练气五层，这是一个重要的门槛。达到这个境界，就有资格参加内门选拔了。

"终于到了。"李凡长舒一口气，脸上露出笑容。这三天，他几乎没怎么休息，一直在做任务、修炼。现在，终于有了回报。

双倍返利系统解锁进度：百分之八十
解锁条件：成为任务堂执事，修为要求练气五层
当前修为：练气五层，已达标百分之百
剩余条件：通过任务堂考核

"还需要通过任务堂考核。"李凡点点头，眼中闪过一丝思索。

这很正常。如果光有修为就能当执事，那太简单了。天下哪有那么多好事。

"先去报名内门选拔。"他整理了一下衣服，迈步向外走去。阳光洒在身上，温暖而明亮。

// 报名选拔

李凡来到外门执事堂，找到王执事。执事堂内庄严肃穆，檀香袅袅。

"王执事，我要报名内门选拔。"李凡躬身说道，态度恭敬。

王执事抬头看了他一眼，眼中闪过一丝惊讶。他放下手中的笔，仔细打量李凡。

"你要报名？"王执事问道，语气中带着一丝疑惑。

"是的。"李凡点点头，神色坚定。

"你现在的修为是……"王执事探查了一下李凡的修为，眼中震惊之色更浓，"练气五层？"

他站起身，绕着李凡走了一圈，上下打量。

"三天时间，从练气四层突破到五层？"王执事难以置信，摇了摇头。

"侥幸而已。"李凡谦虚道，微微低头。

王执事盯着他看了半天，最后说道："行，我给你报名。"他拿起笔，在名册上写下李凡的名字。

"内门选拔在下个月初一，还有十五天。"王执事放下笔，继续说道，"到时候，外门所有练气五层以上的弟子，都可以参加。"

"选拔方式：擂台战。"他顿了顿，"最后胜出的前十名，可以成为内门弟子。"

"明白了。"李凡点点头，将信息记在心中。

"还有……"王执事顿了顿，声音压低，眼中闪过一丝担忧，"赵虎也报名了。"

李凡心中一凛，眉头微皱。

"赵虎？"他低声问道，眼中闪过一丝警惕。

"是的。"王执事冷笑一声，"他早就想进内门了，但修为一直卡在练气五层。"

他凑近李凡，声音更低："听说最近他得到了什么奇遇，修为突破到了练气六层。这次选拔，他是热门人选。"

王执事拍了拍李凡的肩膀，语重心长："你……小心点。"

李凡点点头，拱手道："多谢执事提醒。"

他离开执事堂，心中有了警惕。阳光照在脸上，却驱不散他心中的阴霾。

赵虎修为比他高一层，实力不容小觑。而且，赵虎对他怀恨在心，肯定会在选拔上报复。

"看来，得好好准备了。"李凡握紧拳头，眼中闪过一丝坚定。

---

// 修炼功法

回到小屋后，李凡开始思考如何提升实力。他坐在床沿，双手撑着下巴，陷入沉思。

"修为是练气五层，但赵虎是练气六层。"他低声分析，"硬拼肯定不行，得靠技巧。"

他取出《青云诀》，这是青云宗的基础功法。书页泛黄，边角有些磨损，显然被很多人翻阅过。

每个弟子都能学到，但大多数人只学个皮毛。李凡不一样。

他有系统，可以查看功法的详细信息。但他没有问系统。因为免费额度用完了，他不想消耗寿元。

"那就自己研究。"李凡深吸一口气，翻开书页。

这本书他很熟悉，原主修炼了三年，早就滚瓜烂熟。但以前，他只是按部就班地修炼，没有深入理解。

现在，他静下心来，慢慢琢磨。窗外传来鸟鸣声，清脆悦耳。

"青云诀的第一层，是引气入体。"他轻声念叨，"第二层，是炼化灵气。第三层，是运转周天。第四层是……"

李凡一页一页地看下去，眼睛一眨不眨。阳光渐渐西斜，屋内的光线变得柔和。

突然，他发现了一个细节。手指停在书页上，微微颤抖。

"这里不对。"他皱起眉头，眼中闪过一丝疑惑。

"按照书上的方法，灵力运转到丹田时，应该停顿三息。"他用手指在丹田位置比划，"但如果停顿两息，会怎么样？"

李凡心中一动，开始尝试。他闭上眼睛，盘膝而坐，运转灵力。

灵力在体内流转，当到达丹田时，他只停顿了两息。呼吸变得深沉而缓慢。

轰！

一股强大的灵力从丹田涌出，比平时多了三成！那灵力如潮水般汹涌，让他浑身一震。

"成功了！"李凡心中狂喜，眼睛猛地睁开，眼中满是兴奋。

"这是……功法优化？"他喃喃自语，"原版的《青云诀》，可能有错误，或者不是最优解。而我，找到了更好的修炼方法！"

李凡继续尝试，修改其他地方的运转方式。他完全沉浸在修炼中，忘记了时间的流逝。

一个时辰后，他总结出了新的修炼方法：

停顿时间：三息改两息
运转路线：直线改螺旋
灵力压缩：增加一成

按照这个方法修炼，效率提升了五成！李凡感受着体内充沛的灵力，脸上露出满意的笑容。

"太好了！"他兴奋地说道，"这样下去，我的修炼速度会远超其他人。十五天后，说不定能突破到练气六层！"

李凡很兴奋。但他不知道的是，《青云诀》是青云宗的镇宗功法之一。无数前辈高人修炼过，都没发现问题。

而他，一个穿越者，却找到了优化方法。这说明了什么？

说明他的"全知任务"系统，可能还有其他功能。比如……功法分析？

"系统，你有没有功法分析功能？"李凡在心中问道，眼中带着期待。

【检测到宿主询问】
【免费额度：三比三】
【功能列表：功法分析，未解锁】
【解锁条件：成为任务堂执事】

"又是执事……"李凡无奈地摇摇头，苦笑道。

看来，很多功能都要成为执事后才能解锁。他叹了口气，将《青云诀》收好。

"那就只能靠我自己了。"李凡继续修炼。他不想再消耗寿元了。虽然有了《寿元诀》的想法，但还是要省着用。

窗外的天色渐渐暗了下来，屋内点起了油灯。昏黄的灯光下，李凡的身影显得格外坚定。

// 赵虎的阴谋

就在李凡修炼的时候，外门某个小院里。院内种着几棵桃树，此时桃花盛开，香气四溢。

赵虎正和一个中年男人密谈。两人坐在石桌旁，神色严肃。

"虎儿，那个李凡，报名内门选拔了。"中年男人说道，眼中闪过一丝阴狠。他是赵虎的父亲，赵无极。

"我知道。"赵虎冷笑一声，嘴角勾起一抹不屑，"他一个练气五层的废物，也敢参加选拔？"

他端起茶杯，轻轻抿了一口，动作优雅却带着几分傲慢。

"你……确定他是练气五层？"中年男人问，眉头微皱。

"我亲自探查过，没错。"赵虎放下茶杯，眼中闪过一丝嫉妒，"三天时间，从练气四层突破到五层，肯定是有什么奇遇。"

"所以，你要在选拔上……"中年男人欲言又止，眼中闪过一丝询问。

"放心，父亲。"赵虎眼中闪过一丝狠厉，手指轻轻敲击桌面，"我已经在擂台上做了手脚。"

"什么手脚？"中年男人身体前倾，眼中带着好奇。

"我在自己的擂台下面，布置了一个阵法。"赵虎得意地笑，笑声中带着几分阴险，"只要李凡敢上台，我就激活阵法。"

他顿了顿，继续说道："到时候，他的灵力会被压制三成。而我，会得到三成增幅。一增一减，就是六成的差距！"

赵虎眼中满是自信："就算他有什么奇遇，也赢不了我！"

中年男人点点头，脸上露出满意的笑容："做得好。"

他顿了顿，又叮嘱道："不过，你要小心，别被人发现了。"

"父亲放心，阵法很隐蔽，没人能发现。"赵虎摆摆手，一脸不在乎，"再说了，就算被发现，也可以推给'擂台年久失修'。"

"哈哈，好！"父子俩相视一笑，眼中满是阴谋。笑声在院中回荡，显得格外刺耳。

他们不知道的是，墙角有个黑影，听到了他们的对话。那黑影如同一只猫，悄无声息地趴在墙头。

"赵虎作弊……"黑影低声念叨，眼中闪过一丝思索。

"这件事，得告诉李凡。"

黑影悄无声息地离开了，如同来时一样，没有留下任何痕迹。月光洒在墙头，一片清冷。

// 神秘人

第二天，李凡正在修炼，有人敲门。敲门声不大，但很急促。

"谁？"李凡睁开眼，问道，声音中带着一丝警惕。

"是我，刘大海。"门外传来一个熟悉的声音。

李凡打开门，看到刘大海站在门口。刘大海神色有些紧张，额头上渗着汗珠。

"刘师兄，有事？"李凡问道，侧身让开。

"有件重要的事，要告诉你。"刘大海压低声音，左右看了看，"能进去说吗？"

"请进。"李凡把刘大海让进屋里，顺手关上门。

刘大海坐下后，端起茶杯喝了一口，平复了一下情绪。

"昨晚，我无意中听到赵虎父子的对话。"他放下茶杯，神色严肃，"他们……在选拔上做了手脚。"

"什么手脚？"李凡心中一凛，身体前倾，眼中闪过一丝警惕。

"赵虎在自己的擂台下面，布置了一个阵法。"刘大海说道，声音压得更低，"可以压制对手的灵力，增幅自己的灵力。"

他顿了顿，继续说道："如果你抽到和他在一个擂台，会很吃亏。"

李凡沉默了，眉头紧锁，手指不自觉地敲击着桌面。

"这个消息，可靠吗？"他问道，眼中带着审视。

"我亲耳听到的。"刘大海认真地说，"赵虎和他父亲密谋，我躲在墙后，听得清清楚楚。"

"多谢刘师兄告知。"李凡真诚地说，拱手行礼。

"不用谢。"刘大海摆摆手，眼中闪过一丝正义，"我就是看不惯赵虎那小人。"

他顿了顿，继续说道："对了，我给你一个建议。"

"什么建议？"李凡问道，眼中带着期待。

"选拔是抽签决定擂台的。"刘大海说道，"如果你抽到和赵虎一个擂台，可以申请换擂台。"

"规则允许吗？"李凡皱眉问道。

"允许的。"刘大海点点头，"只要双方都同意，就可以换。"

他叹了口气，摇摇头："但赵虎肯定不会同意。所以……"

刘大海顿了顿，神色凝重："你得做好心理准备。如果真的对上他，要想办法破解阵法。"

"我明白了。"李凡点点头，将信息记在心中。

"那我先走了，你自己小心。"刘大海站起身，告辞离开。

刘大海离开后，李凡陷入了沉思。他走到窗前，望着窗外的景色，心中思绪万千。

"赵虎作弊……"他低声念叨，"这件事，要不要揭发？"

"如果揭发，赵虎肯定会被取消资格。"他继续分析，"但问题是，没有证据。"

"刘大海只是听到，没有亲眼看到。而且，赵虎父子肯定不会承认。"

李凡握紧拳头，眼中闪过一丝决然："所以，只能靠我自己。"

"阵法又如何？"他转身看向屋内，眼中燃起斗志，"我就不信，破不了！"

阳光透过窗户洒在他身上，为他镀上一层金色的光晕。

// 阵法知识

要破解阵法，首先要懂阵法。李凡下定决心，立刻行动。

他来到外门藏书阁，寻找阵法相关的书籍。藏书阁内书香弥漫，一排排书架整齐排列。

"管理员，我想借阵法入门的书。"李凡走到柜台前，恭敬地说道。

"阵法类在三层，你自己去找吧。"管理员头也不抬地说，继续翻看手中的书。

李凡来到三层，找到了阵法区。这里有很多阵法书籍，从入门到精通，应有尽有。书卷的气息扑面而来。

"先看入门的。"李凡取出一本《阵法基础》，开始阅读。书页泛黄，字迹清晰。

阵法，是修仙界的重要技能之一。它可以用来攻击、防御、困敌、辅助等等。

但学习阵法，需要很高的天赋。大多数人，一辈子都学不会。李凡深知这一点，但他没有退缩。

李凡快速浏览，寻找有用的信息。他的目光如饥似渴，生怕错过任何一个细节。

很快，他找到了一段话，眼睛一亮：

"阵法的核心，是灵力节点。每个阵法，都有至少一个灵力节点。破坏节点，阵法就会失效。"

"灵力节点……"李凡心中一动，手指轻轻敲击书页。

"如果能找到赵虎布置的阵法的节点，就能破解。"他继续思考，"但问题是，我不知道阵法的具体位置。"

"除非……"李凡想到了一个办法，眼中闪过一丝希望。

"系统，有没有阵法探测功能？"他在心中问道。

【检测到宿主询问】
【免费额度：三比三】
【功能列表：阵法探测，未解锁】
【解锁条件：成为任务堂执事】

"又是执事……"李凡无奈地摇摇头。

看来，很多功能都要成为执事后才能解锁。他叹了口气，将注意力重新放回书本上。

"那就只能靠我自己了。"李凡继续看书，学习阵法知识。他看得很认真，恨不得把所有知识都记在脑子里。

因为他知道，这可能是他破解赵虎阵法的关键。时间一分一秒地过去，他完全沉浸在知识的海洋中。

一天下来，李凡看了十本阵法书。虽然不能成为阵法大师，但基本的知识已经掌握了。

他合上最后一本书，伸了个懒腰，骨头发出咔咔的声响。

"明天，去选拔现场看看。"他自言自语，"提前熟悉环境，也许能找到阵法的线索。"

李凡合上书本，离开了藏书阁。夜幕降临，青云宗陷入了寂静。月光如水，洒在青石路上。

但李凡的心中，却燃起了斗志。那斗志如同火焰，熊熊燃烧。

"赵虎，选拔台上见。"他望着夜空，低声说道。

"这一次，我要堂堂正正地赢你！"

而在远处的一座山峰上，一个身影望着外门的方向，喃喃自语。那身影笼罩在夜色中，看不清面容。

"内门选拔……"身影低声念叨，声音中带着一丝期待。

"李凡……希望你能走得更远。因为，你可能是唯一能改变这一切的人。"

身影转身，消失在夜色中。只留下一阵微风，吹过山巅。
