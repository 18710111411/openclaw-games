## 📈 伏笔管理规则

### 铺设原则

1. **早期铺设** - 前 100 章铺设所有伏笔
2. **自然融入** - 不刻意，融入剧情
3. **多次暗示** - 每个伏笔至少暗示 3 次
4. **记录在案** - 每次铺设更新此表

### 回收原则

1. **高潮回收** - 在卷末或高潮处回收
2. **意外性** - 回收方式要出乎意料
3. **闭环** - 回收后形成完整逻辑链
4. **更新状态** - 回收后立即更新此表

### 检查清单

每章写完后检查：
- [ ] 是否有新伏笔铺设？
- [ ] 是否有伏笔需要回收？
- [ ] 伏笔回收是否合理？
- [ ] 是否需要更新此表？
