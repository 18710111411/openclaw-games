# 修仙法术体系

> 《修仙：我发布任务，玩家双倍返利》专用修仙设定

---

## 一、灵根体系

### 1.1 灵根类型

| 灵根 | 品质 | 修炼速度 | 战力 | 说明 |
|------|------|---------|------|------|
| **天灵根** | 极品 | 100% | 100% | 单属性，修炼无瓶颈，百年内必化神 |
| **地灵根** | 上品 | 70% | 90% | 双属性，修炼较快，两百年内可化神 |
| **玄灵根** | 中品 | 50% | 80% | 三属性，修炼一般，五百年内可化神 |
| **黄灵根** | 下品 | 30% | 70% | 四属性，修炼缓慢，千年内可化神 |
| **杂灵根** | 废品 | 10% | 50% | 五属性俱全，修炼极慢，基本无望化神 |

### 1.2 属性分类

**五行属性：**
- 金灵根：主杀伐，攻击力最强
- 木灵根：主生机，恢复力最强
- 水灵根：主柔韧，防御力最强
- 火灵根：主爆裂，爆发力最强
- 土灵根：主厚重，耐力最强

**变异属性（稀有）：**
- 风灵根：由木 + 火变异，速度快
- 雷灵根：由金 + 火变异，攻击力极强
- 冰灵根：由水 + 风变异，控制力强
- 阴灵根：由水 + 土变异，擅长隐匿
- 阳灵根：由火 + 土变异，克制阴邪

### 1.3 本书主角设定

| 角色 | 灵根 | 品质 | 说明 |
|------|------|------|------|
| **李凡** | 四灵根（金木水火） | 黄灵根 | 修炼慢，但经验丰富 |
| **林清雪** | 冰灵根（变异） | 天灵根 | 修炼快，战力强 |
| **郝运** | 五灵根 | 杂灵根 | 修炼极慢，但运气爆棚 |
| **赵大牛** | 土灵根 | 地灵根 | 防御强，忠诚可靠 |

## 三、法术体系（分境界版）

### 3.1 法术品阶总览

**法术品阶与境界对应：**

| 境界 | 可用法术品阶 | 威力系数范围 | 灵力上限 |
|------|------------|------------|---------|
| **练气期** | 凡阶（下/中/上品） | 0.5x - 1.0x | 100-500 |
| **筑基期** | 黄阶（下/中/上品） | 1.5x - 2.5x | 500-2000 |
| **金丹期** | 玄阶（下/中/上品） | 3.5x - 7.0x | 2000-10000 |
| **元婴期** | 地阶（下/中/上品） | 10x - 20x | 10000-50000 |
| **化神期** | 天阶（下/中/上品） | 30x - 50x | 50000-200000 |

**说明：**
- 威力系数：相对于凡阶上品基础威力
- 灵力上限：该境界最大灵力值

### 5.3 战斗流程（详细版）

#### 阶段 1：探查阶段（3-5 秒）

**行动：**
- 观察对手境界（灵力波动）
- 分析对手灵根（法术属性）
- 评估战力差距（快速计算）
- 扫描周围环境（地形/障碍物）

**心理活动：**
```
李凡心中快速盘算："对方练气 6 层，比我高 3 层，
但我是四灵根，法术多样，应该有一战之力。"
```

---

#### 阶段 2：准备阶段（5-10 秒）

**行动：**
- 开启防御功法（金钟罩/冰甲术）
- 准备攻击法术（火球术/冰锥术）
- 占据有利地形（高处/掩体后）
- 检查灵力储备（是否充足）

**示例：**
```
李凡双手快速结印，口中默念："金钟罩！"
一层淡金色的光罩瞬间笼罩全身。
他脚下轻点，身形向后跃出 5 米，
背靠一块巨石，占据有利地形。
```

---

#### 阶段 3：试探阶段（10-30 秒）

**行动：**
- 使用低消耗法术试探（小火球术/水箭术）
- 观察对手反应（闪避/格挡/反击）
- 寻找破绽（防御空隙/施法间隔）
- 计算对手 CD（冷却时间）

**示例：**
```
"火球术！"
李凡右手一推，一枚火球呼啸而出。
对方侧身闪避，火球擦身而过。
"闪避不错，但反应时间 0.5 秒，有破绽！"
李凡心中暗喜。
```

---

#### 阶段 4：决战阶段（30 秒 -3 分钟）

**行动：**
- 使用中高阶法术（火球术→烈焰风暴）
- 释放连招（1.5x-3.0x 伤害）
- 使用杀手锏（底牌/特殊能力）
- 决出胜负

**示例：**
```
"就是现在！火球术三连发！"
李凡连续推出三次，三枚火球呈品字形飞出。
对方仓促间撑起护盾，但李凡真正的杀招在后头。
"金刃术！"
三柄金色利刃从火球后方射出，
趁对方抵挡火球时，金刃直击要害！
```

---

#### 阶段 5：收尾阶段（10-30 秒）

**行动：**
- 确认对手状态（昏迷/逃跑/投降）
- 收获战利品（储物袋/法宝/丹药）
- 恢复灵力（打坐/丹药）
- 分析战斗得失（经验总结）

**示例：**
```
李凡走上前，确认对方已昏迷。
他熟练地搜出储物袋，检查战利品。
"下品灵石 50 块，黄阶功法一本，还不错。"
他盘腿坐下，取出回灵丹服下，
开始恢复消耗的灵力。
```

### 5.4 战斗描写要点（写作规则）

**必须包含的描写：**

| 描写类型 | 要求 | 示例 |
|---------|------|------|
| **灵力波动** | 每次施法前描写 | "周身灵力涌动，火红色光芒闪烁" |
| **结印动作** | 复杂法术需要 | "双手快速结印，口中默念咒语" |
| **法术形态** | 视觉效果 | "一枚直径半米的火球呼啸而出" |
| **命中效果** | 伤害表现 | "火球命中胸口，炸出一团血雾" |
| **心理活动** | 战术思考 | "他心中快速盘算，对方 CD 还有 3 秒" |
| **环境互动** | 地形影响 | "火球击中地面，炸出一个大坑" |

**禁止出现的描写：**
- ❌ 简单带过（"他一招打败了对方"）
- ❌ 数值化（"造成 500 点伤害"）
- ❌ 游戏界面（"系统提示：战斗胜利"）

**正确示例：**
```
❌ 错误：
李凡用火球术打败了敌人，造成 500 点伤害。

✅ 正确：
李凡右手一推，周身灵力涌动，
一枚直径半米的火球呼啸而出。
火球正中敌人胸口，炸出一团血雾，
焦糊味瞬间弥漫开来。
敌人惨叫一声，倒飞出去，
重重摔在地上，再也爬不起来。
```

### 5.4 本书战斗设定

**李凡战斗风格：**
- 特点：谨慎为主，不冒险
- 常用战术：防御反击，消耗战
- 优势：经验丰富，知道对手弱点
- 劣势：修为提升慢

**林清雪战斗风格：**
- 特点：强势进攻，速战速决
- 常用战术：冰系控制，连续压制
- 优势：天灵根，修为高
- 劣势：经验不足

**郝运战斗风格：**
- 特点：乱打一气，运气爆棚
- 常用战术：无（经常误打误撞）
- 优势：运气好，经常意外获胜
- 劣势：修为低，技术差
