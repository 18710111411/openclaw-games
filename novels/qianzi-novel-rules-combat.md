# 【修仙法术打斗体系】

> 《修仙：我发布任务，玩家双倍返利》专用打斗规则

---

## 📜 总纲

**核心理念：**
- 法术打斗要符合境界设定
- 不越级挑战（除非有特殊原因）
- 描写要细腻，有来有回
- 法术名称要有意境，不直白

---

## 一、功法等级

### 1.1 功法品阶

**四品十二阶：**

**天品**（元婴期及以上可修炼）
- 天品上阶：蕴含法则之力，可引动天地异象
- 天品中阶：接近法则，威力惊人
- 天品下阶：触及法则边缘

**地品**（金丹期可修炼）
- 地品上阶：金丹圆满可修炼
- 地品中阶：金丹中期可修炼
- 地品下阶：金丹初期可修炼

**玄品**（筑基期可修炼）
- 玄品上阶：筑基圆满可修炼
- 玄品中阶：筑基中期可修炼
- 玄品下阶：筑基初期可修炼

**黄品**（练气期可修炼）
- 黄品上阶：练气 7-9 层可修炼
- 黄品中阶：练气 4-6 层可修炼
- 黄品下阶：练气 1-3 层可修炼

---

### 1.2 功法作用

**功法不是技能书，而是修炼方法：**

- **低级功法：** 修炼速度慢，晋级概率低
- **高级功法：** 修炼速度快，晋级概率高（因为与天道运行暗合）

**示例：**
```
《青云诀》- 黄品中阶
- 青云宗基础功法，外门弟子人人可学
- 修炼速度慢，但胜在平稳安全
- 最高可修炼至筑基期

《太玄真经》- 地品上阶（主角专属）
- 上古传承，金丹期可修炼
- 修炼速度极快，但需要特殊体质
- 可修炼至元婴期
```

## 二、法术分类

### 2.1 法术类型

**攻击类法术：**
- 金系：飞剑术、金刃术、庚金指
- 木系：藤缠术、木刺术、青木诀
- 水系：水龙术、冰锥术、玄冰诀
- 火系：火球术、烈焰术、焚天诀
- 土系：土刺术、山崩术、厚土诀

**防御类法术：**
- 护体罡气
- 五行护盾
- 金光咒
- 玄水盾

**辅助类法术：**
- 隐身术
- 遁术（金木水火土五行遁）
- 飞行术
- 探查术

**特殊类法术：**
- 搜魂术（禁术）
- 夺舍术（魔道）
- 血祭术（魔道）

---

### 2.2 法术释放流程

**标准流程：**
```
1. 掐诀（手印）
2. 念咒（口诀）
3. 运灵力（经脉运行）
4. 释放（法术成型）
```

**示例描写：**
```
李凡双手快速掐诀，口中念念有词：
"天地玄宗，万炁本根……"

体内灵力顺着经脉快速运转，
在丹田处凝聚成一团金色光球。

他猛地睁开双眼，双手向前一推：
"金光咒，去！"

金色光球脱手而出，在空中化作一道光柱，
直奔对手而去。
```

---

### 2.3 法术描写要点

**每场打斗必须包含：**
- [ ] 法术名称（有意境，不直白）
- [ ] 掐诀动作描写
- [ ] 灵力运行描写
- [ ] 法术效果描写（视觉/听觉/嗅觉/触觉）
- [ ] 对手反应描写
- [ ] 环境变化描写

**示例：**
```
❌ 错误（干巴巴）：
李凡用火球术攻击对方。
对方用护盾挡住。

✅ 正确（细腻描写）：
李凡右手掐出离火诀，指尖腾起一簇赤红色火焰。
他手腕一抖，火焰在空中化作一条三丈长的火蛇，
张开血盆大口，带着灼热的气息扑向对方。

"玄水盾！"对方脸色微变，双手快速结印。
一面湛蓝色的水盾凭空出现，挡在身前。

火蛇撞在水盾上，发出"滋滋"的声响，
水蒸气弥漫开来，空气中顿时充满了湿热的气息。

水盾表面泛起阵阵涟漪，但终究没有破碎。
"好强的火系法术！"对方眼中闪过一丝凝重。
```

## 三、境界战力体系

### 3.1 境界差距

**境界压制规则：**

| 境界差 | 压制效果 | 说明 |
|--------|---------|------|
| **同境界** | 无压制 | 看功法、法宝、战斗技巧 |
| **小境界差**（如练气 3 层 vs 练气 5 层） | 轻微压制 | 灵力总量差距约 20% |
| **大境界差**（如练气 vs 筑基） | 绝对压制 | 灵力质量不同，无法越级 |
| **超大境界差**（如筑基 vs 金丹） | 碾压 | 一招可秒杀 |

---

### 3.2 各境界战力特点

**练气期：**
- 特点：灵力外放，可施展低阶法术
- 攻击方式：法术为主，近战为辅
- 移动方式：轻功、低空飞行
- 寿元：100 年

**筑基期：**
- 特点：真气化液，灵力质量提升
- 攻击方式：法术威力翻倍，可御剑飞行
- 移动方式：御剑飞行
- 寿元：200 年
- **兽妖设定：** 开启灵智，能理解人言，不能说话

**金丹期：**
- 特点：凝结金丹，灵力浩瀚
- 攻击方式：可施展地品法术，引动天地之力
- 移动方式：遁光飞行
- 寿元：500 年
- **兽妖设定：** 可以说人话

**元婴期：**
- 特点：元婴出窍，第二条命
- 攻击方式：可施展天品法术，触及法则
- 移动方式：瞬移、虚空穿梭
- 寿元：1000 年
- **兽妖设定：** 可以化为人形

---

### 3.3 战力对比示例

**练气期对战：**
```
练气三层 vs 练气三层：
- 看功法品阶（黄品下阶 vs 黄品中阶）
- 看法宝（无 vs 低阶法器）
- 看战斗技巧

练气三层 vs 练气五层：
- 灵力总量差距约 40%
- 低境界几乎无法战胜高境界
- 除非有逆天法宝或功法
```

**筑基期对战：**
```
筑基初期 vs 筑基初期：
- 看功法（玄品 vs 玄品）
- 看法宝（法器 vs 法宝）
- 看战斗经验

筑基初期 vs 筑基后期：
- 灵力质量差距明显
- 真气化液程度不同
- 几乎无法越级挑战
```

**金丹期对战：**
```
金丹初期 vs 金丹初期：
- 看功法（地品 vs 地品）
- 看法宝（金丹级法宝）
- 看对天地之力的引动

金丹初期 vs 金丹后期：
- 金丹凝实度不同
- 引动天地之力的程度不同
- 无法越级
```

## 四、法宝体系

### 4.1 法宝等级

**凡器：** 练气期使用
- 下品凡器：普通兵器
- 中品凡器：精炼兵器
- 上品凡器：接近法器

**法器：** 筑基期使用
- 下品法器：可注入灵力
- 中品法器：有特殊效果
- 上品法器：接近法宝

**法宝：** 金丹期使用
- 下品法宝：蕴含部分法则
- 中品法宝：法则完整
- 上品法宝：接近通天灵宝

**通天灵宝：** 元婴期使用
- 蕴含完整法则
- 可引动天地异象

**仙器：** 渡劫期使用
- 蕴含仙道法则
- 可破碎虚空

---

### 4.2 法宝描写要点

**法宝不是武器，是"法"的载体：**

**示例：**
```
❌ 错误（像武器）：
李凡拿出一把飞剑，砍向对方。

✅ 正确（法宝描写）：
李凡张口一吐，一柄青色飞剑从口中飞出。
剑身长三尺三寸，通体晶莹，隐隐有风雷之声。

"去！"他手指一点。
飞剑化作一道青色流光，直刺对方咽喉。

飞剑所过之处，空气被撕裂，发出尖锐的破空声。
那是风之法则在运转的征兆。
```

---

### 4.3 法宝温养设定

**法宝需要温养：**
- 平时收入体内（丹田/识海）
- 用灵力温养，增强联系
- 温养时间越长，威力越大

**示例：**
```
"这是……本命法宝？"对方脸色大变。

李凡点点头，嘴角勾起一抹笑意。
"此剑我已温养十年，与我心意相通。"

他心念一动，飞剑在空中一个回旋，
回到他身边，发出亲昵的剑鸣声。
```

## 五、打斗描写规则

### 5.1 打斗节奏

**标准打斗流程：**
```
1. 对峙（气势交锋）
2. 试探（试探对方实力）
3. 交锋（法术对轰）
4. 高潮（底牌尽出）
5. 结局（分出胜负）
```

---

### 5.2 打斗描写要点

**每场打斗必须包含：**
- [ ] 气势描写（眼神/气息/威压）
- [ ] 法术名称和效果
- [ ] 灵力运行描写
- [ ] 环境变化（被打斗影响）
- [ ] 围观者反应（如有）
- [ ] 心理活动（双方想法）

---

### 5.3 打斗示例

**完整打斗示例：**

```
李凡与赵无极对峙在擂台上。

秋风瑟瑟，卷起几片落叶，在空中打着旋儿。

"李凡，今日就是你的死期。"
赵无极冷笑一声，周身元婴初期的气息毫无保留地释放出来。
那气息如同实质的浪潮，压得台下弟子喘不过气来。

李凡神色平静，波澜不惊。
他体内金丹缓缓旋转，金色灵力顺着经脉流淌。
"赵长老，动手吧。"

"找死！"
赵无极怒喝一声，双手快速掐诀。
"血海滔天！"

他身后凭空出现一片血海，腥臭味弥漫开来。
血海中，无数冤魂哀嚎，让人毛骨悚然。

"是魔道功法！"台下有长老惊呼。

李凡瞳孔一缩，脸上终于露出凝重之色。
他双手结印，口中念念有词：
"天地玄宗，万炁本根……金光咒，起！"

金色光柱从他头顶冲天而起，
化作一面金色巨盾，挡在身前。

血海撞在金光盾上，发出"滋滋"的声响。
金光盾表面泛起阵阵涟漪，但终究没有破碎。

"有点本事。"赵无极眼中闪过一丝惊讶，
"但也就到此为止了。"

他双手一合，血海翻腾，
化作一条百丈长的血龙，张开血盆大口咬向李凡。

李凡深吸一口气，体内金丹疯狂旋转。
"既然如此……那就让你看看，我的真正实力。"

他右手虚握，一柄青色长剑从虚空中凝聚而出。
剑身长三尺，通体晶莹，隐隐有风雷之声。

"这是……法宝！"台下再次惊呼。

"太玄剑诀，第一式——破军！"

李凡一剑刺出。
青色剑光划破长空，
与血龙撞在一起。

轰！

惊天动地的爆炸声响起，
整个擂台都在颤抖。

烟尘散去，
李凡持剑而立，衣袂飘飘。
赵无极则是单膝跪地，嘴角溢血。

"你……你竟然……"赵无极难以置信地看着李凡。

"我说过，不要小看我。"
李凡收剑入鞘，转身离去。
```

## 六、特殊设定

### 6.1 越级挑战条件

**原则上不允许越级挑战，除非：**
- 有逆天功法（高两个品阶以上）
- 有逆天法宝（高两个等级以上）
- 有特殊体质（如混沌体、先天道体）
- 有系统/金手指加成

**示例：**
```
李凡（金丹初期）vs 赵无极（元婴初期）

正常情况下：李凡必败
实际情况：
- 《太玄真经》地品上阶（高两个品阶）
- 太玄剑（上品法宝，高一个等级）
- 双倍返利系统（灵力总量翻倍）
- 结果：险胜
```

---

### 6.2 灵力恢复

**灵力恢复方式：**
- 打坐恢复（慢）
- 服用回灵丹（中）
- 极品灵石（快）
- 特殊功法（极快）

**示例：**
```
李凡盘腿坐下，从储物袋中取出一块极品灵石。
灵石通体晶莹，散发着浓郁的灵气。

他手握灵石，运转《太玄真经》。
灵石中的灵气如同决堤的洪水，
疯狂涌入他的体内。

短短一炷香时间，
消耗的灵力就恢复了七成。
```

---

### 6.3 伤势设定

**伤势等级：**
- 轻伤：皮外伤，不影响战斗
- 中伤：伤及经脉，战力下降
- 重伤：伤及内脏/金丹，战力大减
- 垂死：濒死状态，随时陨落

**恢复方式：**
- 轻伤：自行恢复/金创药
- 中伤：回春丹/疗伤药
- 重伤：九转还魂丹/特殊功法
- 垂死：元婴出逃/夺舍重生

## 七、写作注意事项

### 7.1 禁止事项

- ❌ 不要出现现代词汇（如"导弹"、"激光"）
- ❌ 不要出现游戏化描述（如"血量"、"蓝条"）
- ❌ 不要越级挑战（除非有特殊原因）
- ❌ 不要秒杀（除非境界差距极大）
- ❌ 不要无脑对轰（要有战术）

---

### 7.2 推荐事项

- ✅ 法术名称要有意境（如"破军"、"惊雷"）
- ✅ 打斗要有来有回（不要一边倒）
- ✅ 要描写环境变化（被打斗影响）
- ✅ 要描写围观者反应（衬托战斗激烈）
- ✅ 要描写心理活动（双方想法）

---

### 7.3 检查清单

**每场打斗后检查：**
- [ ] 法术名称是否有意境？
- [ ] 掐诀/念咒/运灵力描写是否齐全？
- [ ] 五感描写是否齐全（视觉/听觉/嗅觉/触觉/心理）？
- [ ] 境界压制是否合理？
- [ ] 是否有战术（不是无脑对轰）？
- [ ] 环境变化是否描写？
- [ ] 围观者反应是否描写（如有）？
