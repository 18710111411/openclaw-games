# 写作风格指南

> 本书专用写作规范

---

## 📏 字数要求

- **每章：** 5000 字左右
- **浮动范围：** 4500-5500 字
- **不足处理：** 补充细节描写、角色互动
- **超出处理：** 拆分或精简冗余内容

---

## 🚫 禁用格式

### 章节内禁止使用

1. **编号列表** - 不要用"一、二、三"或"1. 2. 3."
2. **项目符号** - 不要用"- "或"* "列表
3. **表格** - 不要用 markdown 表格
4. **代码块** - 不要用 ``` 代码块

### 原因

- 破坏阅读沉浸感
- 像说明书而不是小说
- 读者出戏

### 替代方案

**❌ 错误示例：**
```
李凡想了想，有三个选择：
一、继续发布任务
二、亲自出去探索
三、找老酒鬼商量
```

**✅ 正确示例：**
```
李凡摩挲着下巴，目光在任务堂内扫过。

继续发布任务固然稳妥，但总觉得缺了些什么。亲自出去探索倒是个不错的选择，只是眼下境界还不够。至于找老酒鬼商量...那老头神出鬼没的，谁知道能不能找到。

他叹了口气，手指轻轻敲击着桌面。
```

## 🐌 节奏控制

### 【钳子小说规则】核心

**慢节奏，细描写，丰富心理**

### 核心原则

- 不急于推进主线剧情
- 不急于解决困难/问题
- 不急于回收伏笔
- 注重日常细节
- 注重角色互动

### 具体做法

**1. 对话必须丰富（【钳子小说规则】核心）**

**每句对话必须包含：**
- 心理活动（至少 1 处）
- 表情描写（至少 1 处）
- 动作描写（至少 1 处）

```
❌ 错误（干巴巴对话）：
"任务完成了吗？"李凡问。
"完成了。"弟子回答。
"很好。"李凡点头。

✅ 正确（丰富描写）：
"任务完成了么？"李凡抬眼看了看眼前的年轻弟子，
嘴角微微上扬，露出一抹不易察觉的笑意。
他心里盘算着，这已经是今天第五个完成任务的玩家了，
双倍返利下来，足足多了两万灵石。

"完...完成了！"弟子咽了口唾沫，
双手有些颤抖地将任务物品递上前，
脸上写满了兴奋和紧张。
他心想这位李执事虽然看起来温和，
但总觉得哪里不对劲，仿佛能看穿自己的心思。

"很好。"李凡点点头，
修长的手指在储物袋上一拂，
眼中闪过一丝满意的神色。
"这是你的奖励，拿去吧。"
他心里暗笑，你拿一万灵石就高兴成这样，
却不知道我得了双倍，整整两万啊。
```

**2. 场景描写要细**

```
❌ 快速带过：
他走进任务堂，开始工作。

✅ 细腻描写：
清晨的阳光透过雕花窗棂，斜斜地洒在任务堂的青石地面上。
李凡推开厚重的木门，门轴发出轻微的吱呀声。
空气中弥漫着淡淡的檀香，那是昨夜值夜弟子留下的。
他走到惯常的位置坐下，手指轻轻抚过桌面，感受着木纹的质感。
```

**3. 场景描写要细（五感齐全）**

```
❌ 快速推进：
"任务完成了吗？"
"完成了。"
"好，这是奖励。"

✅ 自然对话：
"李执事，任务完成了。"

年轻弟子将一袋灵石放在桌上，脸上带着掩饰不住的兴奋。

李凡抬眼看了看，不紧不慢地问道："这么快？可遇到什么难处？"

"倒也不算难，就是那几只铁皮兽有些棘手..."弟子挠挠头，开始讲述经过。

李凡静静听着，不时点点头。等弟子说完，才从储物袋中取出奖励，推到他面前。
```

**4. 场景描写示例**

```
❌ 急于回收：
第 1 章埋的伏笔，第 3 章就解释清楚。

✅ 长期铺设：
第 1 章提到神秘玉佩
第 10 章再次出现
第 30 章揭示部分真相
第 100 章完全揭开
```

**5. 伏笔可以长期铺设**

```
❌ 快速解决：
遇到强敌→立刻打败→获得奖励

✅ 慢慢来：
遇到强敌→暂时退避→寻找方法→多次尝试→最终解决
（可以跨越数章甚至数十章）
```

## 🎭 角色塑造

### 伙伴/配角体系（【钳子小说规则】）

#### 核心伙伴

| 角色 | 身份 | 性格 | 出场章节 |
|------|------|------|---------|
| **王师兄** | 内门弟子 | 神秘，沉稳 | 第 8 章起 |
| **赵大牛** | 外门弟子 | 憨厚，忠诚 | 第 15 章起 |
| **钱多多** | 商会少主 | 精明，贪财 | 第 20 章起 |

#### 逗比福星

| 角色 | 身份 | 特点 | 出场章节 |
|------|------|------|---------|
| **郝运** | 外门弟子 | 逗比，运气爆棚 | 第 18 章起 |
| **贾正经** | 执事堂弟子 | 装逼，爱吹牛 | 第 22 章起 |

---

### 对话风格

每个角色要有独特的说话方式：

| 角色 | 说话风格 | 示例 |
|------|---------|------|
| 李凡 | 沉稳，偶尔吐槽 | "这个我熟，当年我肝了 XXX 次" |
| 我是打工王 | 话痨，兴奋 | "李执事的任务，闭眼接！" |
| 林清雪 | 简洁，高冷 | "嗯。" "走吧。" |
| 老酒鬼 | 神秘，暗示 | "年轻人，有些事急不得..." |

### 心理描写

- 不要过度解释角色想法
- 通过行动和对话展现性格
- 适当留白，让读者自己体会

## 🌍 世界观展现

### 兽妖设定

| 境界 | 能力 | 写作注意 |
|------|------|---------|
| **筑基期** | 开启灵智 | 能理解人言，不能说话，用行动/眼神表达 |
| **金丹期** | 说人话 | 可以口吐人言交流 |
| **元婴期** | 化人形 | 可以化为人形活动 |

**示例：**
```
❌ 错误（筑基期兽妖说话）：
筑基期的铁背熊王开口："你们这些人类..."

✅ 正确（筑基期兽妖用行动表达）：
铁背熊王站起身来，猩红的眼睛盯着李凡，喉咙里发出低沉的咆哮。
它抬起前爪，重重地砸向地面，显然是在警告这些闯入者。
```

---

### 信息揭示规则（【穿越者隐匿法则】）

**核心：** 不要过早透露更早进入游戏的其他穿越者

```
❌ 错误（过早揭示）：
王师兄叹了口气："其实我和你一样，都是穿越者。"

✅ 正确（暗示但不揭示）：
王师兄深深看了李凡一眼，眼神复杂："你和他们...不一样。
有些事，现在知道对你没好处。记住，黑风谷的门，进了就出不来了。"
```

**写作原则：**
- 可以暗示角色知道内情
- 可以暗示角色有秘密
- 不要直接说"他也是穿越者"
- 不要过早揭示穿越者数量和组织

---

### 不要大段说明

```
❌ 错误：
修仙分为炼气、筑基、金丹、元婴、化神...
炼气期可以活 100 年，筑基期 200 年...

✅ 正确：
"张师兄，您这气息...莫非是突破了？"
"侥幸，侥幸。"张长老捋须笑道，"筑基之后，寿元增至两百年，
总算不用像你们这些炼气期的小娃娃，天天担心大限将至。"
```

### 通过剧情自然展现

- 角色对话中透露
- 任务描述中体现
- 战斗过程中展示

## ⚡ 爽点处理

### 不要过于密集

```
❌ 每章都爽：
第 1 章升级→第 2 章打脸→第 3 章获得宝物→第 4 章又升级...

✅ 张弛有度：
第 1 章铺垫→第 2 章发展→第 3 章小高潮→第 4 章日常→第 5 章大高潮
```

### 爽点前要有铺垫

```
爽点前至少 1-2 章铺垫：
- 先被轻视
- 再默默准备
- 最后反转打脸
```

---

## 📖 章节结尾

### 悬念设置

- 不要每章都留大悬念
- 可以是小疑问
- 也可以是平静的结束

**示例：**

```
✅ 小悬念：
李凡望向远方，若有所思。

"明天，该去看看她了。"

✅ 平静结束：
天色渐晚，任务堂的灯火一盏盏亮起。
李凡伸了个懒腰，起身准备离开。
今天就这样吧，明天继续。
```

## 🎯 自查清单

每章写完后检查：

### 【钳子小说规则】核心检查

- [ ] 字数是否达到 5000 字左右？
- [ ] 是否使用了编号列表/项目符号？（如有，删除）
- [ ] 节奏是否过快？（如是，增加细节）
- [ ] 是否有急于解决的困难？（如是，延后）
- [ ] 对话是否自然？
- [ ] 场景描写是否细腻？
- [ ] 角色性格是否一致？

### 描写丰富度检查

- [ ] 对话是否有心理描写？（每句至少 1 处）
- [ ] 对话是否有表情描写？（每句至少 1 处）
- [ ] 对话是否有动作描写？（每句至少 1 处）
- [ ] 场景是否有视觉描写？（颜色/形状/光影）
- [ ] 场景是否有听觉描写？（声音/动静）
- [ ] 场景是否有嗅觉描写？（气味）
- [ ] 场景是否有触觉描写？（质感/温度）

### 角色检查

- [ ] 是否有伙伴/配角出场？
- [ ] 逗比角色是否调节气氛？
- [ ] 兽妖是否符合境界设定？（筑基不说话，金丹说人话，元婴化形）
- [ ] 是否违反【穿越者隐匿法则】？
