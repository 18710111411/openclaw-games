# 章节检查清单

> 每章写完后的自检清单

---

## ✅ 基础检查

### 字数要求

- [ ] 本章字数 ≥ 3000 字
- [ ] 本章字数 ≤ 5000 字
- [ ] 如果不足 3000 字，需要补充内容
- [ ] 如果超过 5000 字，考虑拆分

### 章节结构

- [ ] 开头有吸引力（钩子）
- [ ] 中间有发展（冲突/成长）
- [ ] 结尾有悬念（引导下一章）
- [ ] 章节标题恰当

---

## 📖 内容检查

### 12 项质量指标

| 指标 | 检查项 | 状态 |
|------|--------|------|
| **细节真实** | 地点/物品/行为是否可验证？ | ⚪ |
| **角色一致** | 角色性格/语言/行为符合设定？ | ⚪ |
| **剧情连贯** | 和前后章衔接自然？ | ⚪ |
| **伏笔铺设** | 是否铺设/回收伏笔？ | ⚪ |
| **节奏控制** | 张弛有度，不注水？ | ⚪ |
| **对话自然** | 符合角色身份和情境？ | ⚪ |
| **描写细腻** | 五感细节丰富？ | ⚪ |
| **专业准确** | 修仙设定无漏洞？ | ⚪ |
| **情感真实** | 情绪变化合理？ | ⚪ |
| **场景转换** | 过渡自然？ | ⚪ |
| **信息密度** | 避免冗余？ | ⚪ |
| **读者体验** | 阅读流畅？ | ⚪ |

---

## 🎯 爽点检查

### 本章爽点

- [ ] 是否有爽点？（至少 1 个）
- [ ] 爽点类型：□信息差 □身份差 □成长 □装 X □其他____
- [ ] 爽点是否充分展现？
- [ ] 爽点前是否有铺垫？

### 常见爽点清单

| 爽点类型 | 示例 | 是否使用 |
|---------|------|---------|
| **信息差** | 主角知道攻略，别人不知道 | ⚪ |
| **身份差** | NPC 身份调教玩家 | ⚪ |
| **成长快** | 境界/技能提升 | ⚪ |
| **装 X 打脸** | 被轻视后反转 | ⚪ |
| **幕后操控** | 玩家以为是 X，其实是 Y | ⚪ |
| **收获惊喜** | 获得稀有物品/技能 | ⚪ |
| **玩家脑补** | 玩家过度解读主角行为 | ⚪ |
| **双倍返利** | 触发暴击，奖励翻倍 | ⚪ |

---

## 🎭 角色检查

### 主角表现

- [ ] 主角性格是否符合设定？
- [ ] 主角是否有成长？
- [ ] 主角是否有主动行动？（不是被动挨打）
- [ ] 主角口头禅是否使用？（如适用）

### 配角表现

- [ ] 出场配角是否符合设定？
- [ ] 配角是否有作用？（推动剧情/提供信息）
- [ ] 配角对话是否有辨识度？

### 玩家互动（如适用）

- [ ] 玩家反应是否真实？
- [ ] 玩家是否有吐槽？（增加趣味性）
- [ ] 玩家是否"脑补"？（迪化流）
- [ ] 玩家是否获得合理奖励？

---

## 🌍 世界观检查

### 修仙设定

- [ ] 境界设定是否正确？
- [ ] 功法/技能是否符合体系？
- [ ] 灵石/货币系统是否一致？
- [ ] 宗门/势力设定是否合理？

### 游戏设定

- [ ] 玩家等级/境界对应是否正确？
- [ ] 任务系统是否符合设定？
- [ ] 论坛/聊天等元素是否合理？

---

## 📝 伏笔检查

### 伏笔铺设

- [ ] 本章是否铺设新伏笔？
- [ ] 如有，记录到 `foreshadowing-tracker.md`
- [ ] 伏笔是否自然融入？（不刻意）

### 伏笔回收

- [ ] 本章是否回收伏笔？
- [ ] 如有，更新 `foreshadowing-tracker.md` 状态
- [ ] 回收是否合理？（有铺垫）

---

## ✍️ 文字检查

### 错别字

- [ ] 通读检查错别字
- [ ] 特别注意：修仙术语、人名、地名

### 语句流畅

- [ ] 是否有病句？
- [ ] 是否有重复表达？
- [ ] 是否有歧义？

### 段落节奏

- [ ] 段落长度是否合适？（不过长）
- [ ] 对话是否分段？
- [ ] 场景转换是否空行？
