## 🎬 常用镜头术语

### 景别

| 景别 | 说明 | 用途 |
|------|------|------|
| **大远景** | 展示宏大场景 | 开场、环境介绍 |
| **远景** | 人物在环境中较小 | 展示人物与环境关系 |
| **全景** | 人物全身 | 展示动作、服装 |
| **中景** | 腰部以上 | 对话、互动 |
| **近景** | 胸部以上 | 表情、情绪 |
| **特写** | 面部或局部 | 强调细节、情绪 |
| **大特写** | 眼睛、手等 | 极度强调 |

### 镜头运动

| 运动 | 说明 | 用途 |
|------|------|------|
| **固定** | 镜头不动 | 稳定、对话 |
| **推** | 向前移动 | 强调、进入 |
| **拉** | 向后移动 | 退出、展示环境 |
| **摇** | 左右转动 | 跟随、展示 |
| **移** | 横向移动 | 跟随、转换 |
| **跟** | 跟随主体 | 动作戏 |
| **升/降** | 垂直移动 | 展示高度 |

## 🎯 分镜设计原则

### 1. 镜头数量

- 对话场景：每句台词 1-2 个镜头
- 动作场景：每个动作 1-3 个镜头
- 场景展示：1-3 个镜头（远→中→近）

### 2. 时长分配

| 类型 | 单镜头时长 |
|------|-----------|
| 对话 | 3-5 秒 |
| 动作 | 2-4 秒 |
| 展示 | 4-6 秒 |
| 特写 | 2-3 秒 |

### 3. 节奏控制

- 开场：慢（展示环境）
- 对话：中（自然节奏）
- 高潮：快（快速切换）
- 结尾：慢（情绪沉淀）
