## 🎬 第二部分：AI 提示词生成公式（增强版）

### 核心公式（八大要素）

```
【主体描述】+【服装细节】+【动作/表情】+【场景环境】+【镜头语言】+【光影氛围】+【动态效果】+【风格参考】
```

### 提示词详细程度标准

| 要素 | 最少字数 | 必须包含 |
|------|---------|---------|
| 主体描述 | 30 字 | 年龄/性别/发型/面部特征/体型 |
| 服装细节 | 20 字 | 服装款式/颜色/材质/磨损状态 |
| 动作/表情 | 25 字 | 具体动作/表情细节/情绪状态 |
| 场景环境 | 30 字 | 地点/时间/建筑/道具/背景元素 |
| 镜头语言 | 20 字 | 景别/镜头运动/焦距/角度 |
| 光影氛围 | 25 字 | 光源类型/光线方向/色调/氛围词 |
| 动态效果 | 20 字 | 运动模糊/粒子效果/特殊效果 |
| 风格参考 | 15 字 | 电影感/风格类型/画质 |
| **合计** | **185 字+** | 完整详细 |

### 从分镜到提示词的转换规则

| 分镜字段 | → | 提示词要素 | 转换说明 |
|---------|---|-----------|---------|
| 画面内容 | → | 主体描述 + 场景环境 | 提取主体、服装、环境细节 |
| 角色 | → | 主体描述 | 添加年龄、外貌、服装 |
| 表情/动作 | → | 动作/表情 | 具体化动作描述 |
| 景别 | → | 镜头语言 | 远景/全景/中景/近景/特写 |
| 镜头运动 | → | 镜头语言 | 推/拉/摇/移/跟/固定 |
| 音效/BGM | → | 光影氛围 | 转化为氛围词 |
| 时长 | → | 风格参考 | 影响节奏描述 |

## 🔍 第八部分：质量检查与优化

### 提示词质量检查表

| 检查项 | 标准 | 检查 |
|--------|------|------|
| 主体清晰 | 年龄/服装/外貌明确 | □ |
| 动作具体 | 动作描述不模糊 | □ |
| 场景详细 | 环境有时间/地点/氛围 | □ |
| 镜头准确 | 景别 + 运动都标注 | □ |
| 光影恰当 | 光线类型 + 效果 | □ |
| 风格明确 | 电影感/动画感等 | □ |
| 无矛盾 | 描述不自相矛盾 | □ |
| 字数适中 | 80-200 字（即梦） | □ |

### 常见问题及优化方案

| 问题 | 原因 | 优化方案 |
|------|------|---------|
| 人物不一致 | 描述不够具体 | 固定人物描述模板，每次复用 |
| 动作不流畅 | 动作描述太抽象 | 使用具体动词，分解动作 |
| 场景错误 | 环境描述缺失 | 添加时间/地点/氛围词 |
| 画面畸形 | 提示词有矛盾 | 检查并删除矛盾描述 |
| 风格不统一 | 风格词缺失 | 统一添加"电影质感，8K"等 |
