## 📚 第一部分：主流 AI 视频模型对比

### 模型能力对比

| 模型 | 擅长领域 | 时长限制 | 中文支持 | 推荐用途 |
|------|---------|---------|---------|---------|
| **即梦 Seedance** | 电影感、人物表演 | 5-10 秒 | ✅ 优秀 | **主力**（人像/剧情） |
| **Runway Gen-2** | 电影特效、复杂场景 | 4 秒 | ⚠️ 一般 | 特效镜头、场景转换 |
| **Pika Labs** | 动画风格、动作流畅 | 3 秒 | ⚠️ 一般 | 动漫风格、动态镜头 |
| **SVD** | 真实感、物理准确 | 4 秒 | ⚠️ 一般 | 写实镜头、物体运动 |
| **Luma Dream** | 长镜头、连贯性 | 8 秒 | ⚠️ 一般 | 长镜头、场景漫游 |

### 即梦 Seedance 优势

- ✅ 中文提示词原生支持
- ✅ 人物表情细腻
- ✅ 电影感强
- ✅ 时长较长（5-10 秒）
- ✅ 国内访问速度快

## 📝 第二部分：分镜格式（主流标准）

### 分镜脚本字段

| 字段 | 说明 | 示例 |
|------|------|------|
| **镜号** | 镜头编号 | SC01-001 |
| **场景** | 场景名称 | 外景·青云宗大门·日 |
| **景别** | 远景/全景/中景/近景/特写 | 中景 |
| **镜头运动** | 推/拉/摇/移/跟/固定 | 缓慢推进 |
| **画面内容** | 详细描述画面 | 李凡站在任务堂门口，身穿青色长袍... |
| **角色** | 出场角色及位置 | 李凡（居中）、玩家甲（左侧） |
| **表情/动作** | 角色表情和动作 | 李凡微笑，手指轻点任务面板 |
| **台词/旁白** | 对话或旁白 | "这个任务报酬丰厚，谁来接？" |
| **音效/BGM** | 背景音乐和音效 | 古风 BGM，鸟鸣声 |
| **时长** | 预计时长 | 5 秒 |
| **备注** | 特殊说明 | 需要光线从左侧射入 |

### 分镜模板（Markdown 格式）

```markdown
# 分镜脚本 - 《修仙：我发布任务，玩家双倍返利》

## 第 X 章 - [章节名]

### 场景 1：[场景名称]

| 镜号 | 景别 | 镜头运动 | 画面内容 | 台词 | 时长 |
|------|------|---------|---------|------|------|
| 001 | 远景 | 缓慢推进 | 青云宗全景，云雾缭绕 | （旁白） | 5s |
| 002 | 中景 | 固定 | 李凡站在任务堂门口 | "这个任务..." | 4s |
| 003 | 特写 | 微推 | 李凡手指轻点面板 | - | 3s |
```

### 分镜编号规则

```
SC[场景号]-[镜号]
例：SC01-001 = 第 1 场景第 1 镜
```

## 🎬 第三部分：视频提示词核心公式

### 即梦 Seedance 提示词结构

```
【主体描述】+【动作/表情】+【场景环境】+【镜头语言】+【光影氛围】+【风格参考】
```

### 详细拆解

#### 1️⃣ 主体描述

**人物：**
```
年轻男子，黑色长发束起，身穿青色古装长袍，面容清秀，眼神锐利
```

**要素：**
- 年龄感（年轻/中年/老年）
- 发型发色
- 服装（款式 + 颜色）
- 面部特征
- 气质特点

#### 2️⃣ 动作/表情

```
嘴角微微上扬，右手抬起轻点空中，眼神专注地看着前方
```

**常用动作词：**
- 微笑/冷笑/皱眉/凝视
- 抬手/转身/行走/坐下
- 点头/摇头/侧耳倾听

#### 3️⃣ 场景环境

```
古风建筑内，木质结构，雕花窗棂，阳光透过窗户洒在地面上
```

**要素：**
- 建筑类型（古风/现代/科幻）
- 材质（木质/石质/金属）
- 装饰细节
- 时间（日/夜/黄昏）

#### 4️⃣ 镜头语言

```
中景，镜头缓慢推进，轻微晃动模拟手持感
```

**常用镜头：**
- 远景（展示环境）
- 全景（展示全身）
- 中景（腰部以上）
- 近景（胸部以上）
- 特写（面部/局部）

**镜头运动：**
- 固定（静止）
- 推进（向前）
- 拉远（向后）
- 摇摄（左右）
- 跟随（跟拍）

#### 5️⃣ 光影氛围

```
柔和的侧光，明暗对比明显，空气中漂浮着细小尘埃
```

**光影类型：**
- 自然光（日光/月光）
- 人造光（烛光/灯光）
- 侧光/逆光/顶光
- 柔光/硬光

**氛围词：**
- 温暖/清冷/神秘/压抑
- 朦胧/清晰/梦幻/真实

#### 6️⃣ 风格参考

```
电影质感，中国古风，细腻写实，8K 超高清
```

**风格词：**
- 电影感/纪录片感/动画感
- 中国风/日式/西式
- 写实/写意/唯美
- 4K/8K/超高清

## 🎯 第五部分：提示词优化技巧

### 提升质量的关键词

| 类别 | 关键词 | 效果 |
|------|--------|------|
| **画质** | 8K 超高清、细腻、精细 | 提升清晰度 |
| **光影** | 电影级布光、丁达尔效应、体积光 | 增强氛围 |
| **镜头** | 浅景深、背景虚化、电影感 | 专业感 |
| **风格** | 中国古风、仙侠、唯美 | 风格统一 |
| **动态** | 流畅、自然、细腻 | 动作质量 |

### 避免的词汇

- ❌ 过于抽象（"很美"、"好看"）
- ❌ 矛盾描述（"白天 + 夜晚"）
- ❌ 过于复杂（超过 3 个主体）
- ❌ 模糊词汇（"可能"、"也许"）

### 权重调整

**即梦支持权重语法：**
```
(关键词：1.5) = 提升权重
(关键词：0.8) = 降低权重

示例：
(青色长袍：1.3)，(古风建筑：1.2)，(柔和光线：0.9)
```

## 📊 第六部分：工作流 SOP

### 完整流程

```
小说原文 → 剧本改编 → 分镜拆分 → 提示词生成 → AI 视频 → 后期合成
   ↓           ↓           ↓           ↓           ↓         ↓
1. 提取    2. 格式化   3. 填模板   4. 优化    5. 生成   6. 剪辑
```

### 各阶段详细说明

#### 1️⃣ 小说→剧本

**输入：** 小说章节
**输出：** 剧本格式

**操作：**
- 提取对话
- 简化场景描述
- 添加旁白（如需要）
- 标注情绪/动作

**示例：**
```
原文：
李凡看着眼前的年轻弟子，想起了自己刚入门时的样子。
他微微一笑，说道："任务完成得不错，这是你的奖励。"

剧本：
【场景】任务堂内·日
【人物】李凡、年轻弟子
【动作】李凡微笑，递出奖励
【台词】李凡："任务完成得不错，这是你的奖励。"
【情绪】温和、鼓励
```

#### 2️⃣ 剧本→分镜

**输入：** 剧本
**输出：** 分镜脚本

**操作：**
- 划分镜头（每句台词 1-3 个镜头）
- 确定景别（远/全/中/近/特）
- 设计镜头运动
- 估算时长

#### 3️⃣ 分镜→提示词

**输入：** 分镜脚本
**输出：** AI 视频提示词

**操作：**
- 选择对应模板
- 填充详细描述
- 优化关键词
- 添加权重

#### 4️⃣ AI 视频生成

**输入：** 提示词
**输出：** 视频片段

**操作：**
- 选择模型（即梦为主）
- 输入提示词
- 调整参数（时长/比例等）
- 生成并检查

#### 5️⃣ 后期合成

**输入：** 视频片段
**输出：** 成品漫剧

**操作：**
- 剪辑拼接
- 添加配音
- 添加 BGM/音效
- 添加字幕
- 调色

## 💡 第九部分：常见问题解决

### Q1：人物不一致

**问题：** 不同镜头中主角长得不一样

**解决：**
- 使用固定提示词模板
- 保存成功的人物描述
- 使用参考图功能（如有）
- 记录成功的提示词

### Q2：动作不连贯

**问题：** 前后镜头动作接不上

**解决：**
- 分镜设计时考虑衔接
- 使用过渡镜头
- 后期剪辑处理
- 减少快速切换

### Q3：提示词无效

**问题：** 生成的视频不符合描述

**解决：**
- 简化提示词（去除冗余）
- 调整权重
- 更换关键词
- 尝试不同模型

### Q4：视频时长不够

**问题：** 对话太长，视频太短

**解决：**
- 拆分多个镜头
- 使用循环/慢动作
- 添加过渡镜头
- 剪辑拼接
