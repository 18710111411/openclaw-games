## 使用方法

### 1. 查看剧本
```bash
cat 01-剧本.md
```

### 2. 查看分镜
```bash
cat 02-分镜.md
```

### 3. 使用提示词生成视频
每个提示词文件对应一个镜头，可直接复制到即梦（Seedance）使用：
```bash
cat 03-提示词/SC01-007.txt  # 系统觉醒关键镜头
```

### 4. 批量生成
将所有提示词依次输入即梦，生成 13 个视频片段后剪辑合成。

---

## 提示词公式

每个提示词遵循即梦公式：
```
主体 + 动作 + 场景 + 镜头 + 光影 + 风格
```

- **主体：** 画面核心人物/物体
- **动作：** 主体正在进行的动作
- **场景：** 环境背景描述
- **镜头：** 景别、镜头运动、构图
- **光影：** 光源、光效、色调
- **风格：** 艺术风格、参考作品、画质

---

## 质量标准

✅ 分镜符合主流格式（景别/镜头运动/画面内容/时长）
✅ 提示词符合即梦公式（主体 + 动作 + 场景 + 镜头 + 光影 + 风格）
✅ 每个提示词 100-300 字中文
✅ 13 个镜头覆盖完整叙事（开场→发展→高潮→结尾）

---

## 后续制作流程

1. **视频生成：** 使用即梦 Seedance 逐镜头生成视频
2. **音频制作：** 配音（旁白 + 角色台词）+ 音效 + BGM
3. **剪辑合成：** 将 13 个镜头按顺序剪辑，添加转场
4. **字幕添加：** 添加台词字幕和效果字幕
5. **调色输出：** 统一色调，导出成片
