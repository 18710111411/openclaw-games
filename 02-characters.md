## 二、核心英雄 (5 人)

### 🦸 1. 雷克斯 (Rex)

| 属性 | 设定 |
|------|------|
| **代号** | 雷克斯 (Rex) |
| **姓名** | 雷蒙德·"雷克斯"·卡特 |
| **年龄** | 35 岁 |
| **性别** | 男 |
| **职业** | 军人/战术指挥官 |
| **稀有度** | SSR (初始英雄) |

**外貌特征：**
- 身高：188cm
- 体型：健壮，肌肉发达
- 发型：短寸头，灰白色
- 服装：战术背心，迷彩裤，军靴
- 武器：突击步枪，战术刀
- 特征：左眼疤痕，右臂纹身

**性格特点：**
- 冷静果断
- 责任感强
- 话少行动派
- 保护弱者

**背景故事：**
> 前特种部队少校，病毒爆发时正在执行保护任务。任务失败后，带领少数幸存者建立基地。自责没能保护更多人，把基地视为最后的防线。

**技能：**

| 技能 | 类型 | 效果 | 冷却 |
|------|------|------|------|
| **空袭呼叫** | 主动 | 呼叫空袭，对指定区域造成大量伤害 | 60 秒 |
| **战地指挥** | 被动 | 附近防御塔攻速 +15% | 常驻 |
| **绝境反击** | 被动 | 基地血量<30% 时，全塔伤害 +30% | 常驻 |

**台词：**
```
出场："指挥官，防线交给我。"
空袭："坐标确认，火力覆盖！"
低血量："还不够...我们还能守住！"
胜利："又活过一天。"
失败："抱歉...我尽力了..."
```

### 🩺 2. 艾琳 (Irene)

| 属性 | 设定 |
|------|------|
| **代号** | 艾琳 (Irene) |
| **姓名** | 艾琳·陈博士 |
| **年龄** | 32 岁 |
| **性别** | 女 |
| **职业** | 医生/病毒学家 |
| **稀有度** | SSR |

**外貌特征：**
- 身高：168cm
- 体型：纤细
- 发型：黑色长发，马尾
- 服装：白大褂，防护手套
- 武器：手枪，医疗包
- 特征：眼镜，脖子上挂着 U 盘 (病毒数据)

**性格特点：**
- 理性冷静
- 执着于解药研究
- 外冷内热
- 有秘密 (知道病毒真相)

**背景故事：**
> 前安布雷拉公司病毒研究员，病毒泄露时偷走核心数据逃出。知道病毒的部分真相，但不完全相信公司高层是幕后黑手。暗中研究解药，希望治愈而非毁灭。

**技能：**

| 技能 | 类型 | 效果 | 冷却 |
|------|------|------|------|
| **紧急治疗** | 主动 | 治疗指定英雄/防御塔 50% 血量 | 30 秒 |
| **解毒专家** | 被动 | 英雄免疫中毒/感染效果 | 常驻 |
| **解药研究** | 被动 | (隐藏) 收集配方碎片后解锁真结局 | 剧情 |

**台词：**
```
出场："病毒...我会找到解药的。"
治疗："别动，很快就好。"
研究："数据不对...一定有什么遗漏..."
真相："安布雷拉...你们到底在隐瞒什么？"
```

**隐藏任务线：**
- 第 5 关：发现病毒样本 → 解锁解药线索
- 第 15 关：获取实验室数据 → 配方碎片 1/5
- 第 25 关：拯救研究员 NPC → 配方碎片 2/5
- 第 35 关：潜入军方基地 → 配方碎片 3/5
- 第 38 关：返回安布雷拉 → 配方碎片 4-5/5
- 第 40 关：选择摧毁 or 夺取 → 真结局解锁

## 六、英雄关系网

```
                    指挥官 (玩家)
                        │
        ┌───────────────┼───────────────┐
        │               │               │
      雷克斯          艾琳           老杰克
        │               │               │
        │           秘密关系          爷孙情
        │               │               │
        └──────────┐    │    ┌──────────┘
                   │    │    │
                 影子 ←─┴→ 安娜
                   │         │
              实验体关系   前同事
```

**关键关系：**
- **雷克斯 ↔ 艾琳**：互相尊重，但理念不同 (摧毁 vs 解药)
- **艾琳 ↔ 影子**：都了解安布雷拉内幕，暗中合作
- **老杰克 ↔ 小豆**：爷孙情，老杰克保护小豆
- **影子 ↔ 安娜**：前同事，都曾在军方服役
- **安娜 ↔ 雷克斯**：前上下级，信任关系
