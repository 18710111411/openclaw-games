## 🎮 核心玩法

### 游戏目标
- 保护地图中央的萝卜不被敌人吃掉
- 在敌人路径旁建造防御塔
- 消灭所有波次的敌人

### 核心循环
```
建造防御塔 → 敌人进攻 → 收集金币 → 升级塔 → 下一波敌人
```

### 游戏特色（差异化）
1. **双返利系统** - 类似小说设定，玩家（防御塔）击杀敌人，主角获得双倍金币
2. **穿越者元素** - 防御塔可以是"玩家化身"，有弹幕吐槽
3. **修仙题材** - 塔=修仙者，敌人=妖兽，萝卜=宗门至宝

## 🏰 防御塔设计

### 基础塔（5 种）

| 塔名 | 类型 | 特点 | 造价 |
|------|------|------|------|
| **箭塔** | 单体 | 攻速快，伤害低 | 100 金币 |
| **炮塔** | 范围 | 伤害高，攻速慢 | 200 金币 |
| **冰塔** | 减速 | 降低敌人速度 | 150 金币 |
| **法塔** | 穿透 | 可攻击多个敌人 | 250 金币 |
| **电塔** | 连锁 | 闪电链攻击 | 300 金币 |

### 进阶塔（10 种）
- 通过基础塔升级/组合获得
- 例如：箭塔 + 冰塔 = 冰箭塔（伤害 + 减速）

### 特殊塔（5 种）
- **玩家化身塔** - 穿越者，有弹幕吐槽
- **双倍返利塔** - 击杀金币翻倍
- **召唤塔** - 召唤临时防御单位
- **陷阱塔** - 地面陷阱
- **辅助塔** - 增益周围塔

## 💰 经济系统

### 金币获取
- 击杀敌人：基础金币
- 波次奖励：每波结束奖励
- 利息系统：每 5 波结算利息（最大金币的 10%）
- **双倍返利**：特定塔击杀获得双倍

### 金币消耗
- 建造防御塔
- 升级防御塔
- 使用技能/道具
- 购买 Buff

### 平衡公式
```
单波敌人总血量 = 基础血量 × (1.15 ^ 波次数)
单波金币产出 = 基础产出 × (1.1 ^ 波次数)
```
