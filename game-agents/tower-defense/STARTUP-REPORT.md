## 📋 项目核心规划

### 游戏设定

| 项目 | 内容 |
|------|------|
| **游戏名称** | 萝卜保卫战（暂定） |
| **游戏类型** | 2D 塔防 |
| **美术风格** | 可爱卡通风 |
| **目标平台** | PC + Web + 移动端 |
| **开发周期** | 8-12 周 |
| **目标成本** | ¥0（使用免费资源） |

### 差异化特色

1. **修仙题材** - 塔=修仙者，敌人=妖兽，萝卜=宗门至宝
2. **双倍返利** - 特定塔击杀获得双倍金币
3. **玩家化身** - 穿越者塔，有弹幕吐槽
4. **修仙境界** - 塔有境界提升系统

### 内容规模

| 内容 | 数量 |
|------|------|
| **关卡** | 50 关（4 个主题区域） |
| **防御塔** | 20 种（5 基础 +10 进阶 +5 特殊） |
| **敌人** | 15 种（4 基础 + 特殊 +BOSS） |
| **预计时长** | 8-12 小时完整通关 |
